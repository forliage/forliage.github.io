线性代数04:多项式

在开始之前，我们先做如下约定：

  • $\mathbf{F}$ 表示 $\mathbf{R}$ 或 $\mathbf{C}$。

4.A 复共轭与绝对值

在讨论复系数多项式之前，我们先回顾一些关于复数的基础知识。

定义 4.2：实部、虚部 设复数 $z = a + bi$，其中 $a$ 和 $b$ 均为实数。

  • $z$ 的实部 (real part)，记作 $\text{Re } z$，定义为 $\text{Re } z = a$。
  • $z$ 的虚部 (imaginary part)，记作 $\text{Im } z$，定义为 $\text{Im } z = b$。

定义 4.3：复共轭、绝对值 设 $z = a + bi \in \mathbf{C}$。

  • $z$ 的复共轭 (complex conjugate)，记作 $\bar{z}$，定义为 $\bar{z} = a - bi$。
  • $z$ 的绝对值 (absolute value)，记作 $|z|$，定义为 $|z| = \sqrt{a^2 + b^2}$。

例子 4.4 设 $z = 3 + 2i$。

  • $\text{Re } z = 3$
  • $\text{Im } z = 2$
  • $\bar{z} = 3 - 2i$
  • $|z| = \sqrt{3^2 + 2^2} = \sqrt{9+4} = \sqrt{13}$

命题 4.5：复数的性质 设 $w, z \in \mathbf{C}$。

  • $z + \bar{z} = 2\text{Re } z$
  • $z - \bar{z} = 2(\text{Im } z)i$
  • $z\bar{z} = |z|^2$
  • $\overline{w+z} = \bar{w} + \bar{z}$
  • $\overline{wz} = \bar{w}\bar{z}$
  • $\bar{\bar{z}} = z$
  • $|\text{Re } z| \le |z|$
  • $|\text{Im } z| \le |z|$
  • $|\bar{z}| = |z|$
  • $|wz| = |w||z|$
  • 三角不等式 (triangle inequality): $|w+z| \le |w| + |z|$

证明 (三角不等式): 我们有： $$ \begin{aligned} |w+z|^2 &= (w+z)(\overline{w+z}) \\ &= (w+z)(\bar{w}+\bar{z}) \\ &= w\bar{w} + z\bar{z} + w\bar{z} + z\bar{w} \\ &= |w|^2 + |z|^2 + w\bar{z} + \overline{w\bar{z}} \\ &= |w|^2 + |z|^2 + 2\text{Re}(w\bar{z}) \\ &\le |w|^2 + |z|^2 + 2|w\bar{z}| \quad (\text{因为 } \text{Re}(x) \le |x|) \\ &= |w|^2 + |z|^2 + 2|w||\bar{z}| \\ &= |w|^2 + |z|^2 + 2|w||z| \\ &= (|w| + |z|)^2 \end{aligned} $$ 在不等式 $|w+z|^2 \le (|w|+|z|)^2$ 两端取非负平方根，即得到所要证明的不等式。证毕。

4.B 多项式的系数与带余除法

多项式系数的唯一性 我们通常认为一个多项式由其系数唯一确定。下面的定理严格证明了这一点。

定理 4.7：一个多项式是零多项式，则其所有系数均为0 设 $a_0, \ldots, a_m \in \mathbf{F}$。若对任意 $z \in \mathbf{F}$ 均有 $$ p(z) = a_0 + a_1 z + \cdots + a_m z^m = 0 $$ 则 $a_0 = a_1 = \cdots = a_m = 0$。

证明： 我们用反证法。假设并非所有系数都为0。我们可以改变 $m$ 的可以，假定 $a_m \neq 0$。 取一个特定的 $z$ 值： $$ z = \frac{|a_0|+|a_1|+\cdots+|a_{m-1}|}{|a_m|} + 1 $$ 由于 $|a_m|>0$ 且各项绝对值非负，所以 $z > 1$。 对于我们选取的 $z$，有 $z^j \le z^{m-1}$ 对 $j=0, \ldots, m-1$ 成立。 利用三角不等式，我们有： $$ \begin{aligned} |a_0 + a_1 z + \cdots + a_{m-1}z^{m-1}| &\le |a_0| + |a_1 z| + \cdots + |a_{m-1}z^{m-1}| \\ &= |a_0| + |a_1| |z| + \cdots + |a_{m-1}||z|^{m-1} \\ &\le |a_0| |z|^{m-1} + |a_1| |z|^{m-1} + \cdots + |a_{m-1}||z|^{m-1} \\ &= (|a_0| + |a_1| + \cdots + |a_{m-1}|)|z|^{m-1} \\ &= (|a_m|(z-1))|z|^{m-1} \\ &< |a_m| z |z|^{m-1} = |a_m z^m| \end{aligned} $$ 因此，我们得到 $|a_0 + a_1 z + \cdots + a_{m-1}z^{m-1}| < |a_m z^m|$。 这意味着 $a_0 + a_1 z + \cdots + a_{m-1}z^{m-1} \neq -a_m z^m$。 所以，$p(z) = a_0 + a_1 z + \cdots + a_m z^m \neq 0$。 这与我们的初始假设“对任意 $z \in \mathbf{F}$ 均有 $p(z)=0$”相矛盾。 因此，假设不成立，所有系数必须都为0。证毕。

多项式的带余除法

定理 4.8：多项式的带余除法 (Division Algorithm for Polynomials) 设 $p, s \in \mathbf{P}(\mathbf{F})$ 且 $s \neq 0$。则存在唯一的多项式 $q, r \in \mathbf{P}(\mathbf{F})$ 使得 $$ p = sq + r $$ 并且 $\text{deg } r < \text{deg } s$。

证明： 设 $n = \text{deg } p$ 和 $m = \text{deg } s$。如果 $n < m$，我们取 $q=0, r=p$ 即可。 现在假设 $n \ge m$。我们定义一个线性映射 $$ T: \mathbf{P}_{n-m}(\mathbf{F}) \times \mathbf{P}_{m-1}(\mathbf{F}) \to \mathbf{P}_n(\mathbf{F}) $$ 定义为 $$ T(q, r) = sq + r $$ 我们需要证明 $T$ 是一个同构（即可逆线性映射）。 首先证明 $T$ 是单射的。设 $T(q, r) = 0$。则 $sq+r=0$，即 $sq = -r$。 因为 $\text{deg } r < m$，所以 $\text{deg}(-r) < m$。 而 $\text{deg}(sq) = \text{deg } s + \text{deg } q = m + \text{deg } q$。 如果 $q \neq 0$，则 $\text{deg } q \ge 0$，那么 $\text{deg}(sq) \ge m$。 等式 $sq=-r$ 意味着一个次数 $\ge m$ 的多项式等于一个次数 $< m$ 的多项式，这是不可能的，除非它们都是零多项式。 因此必有 $q=0$，从而 $r=0$。 所以 $\text{null } T = {(0, 0)}$，$T$ 是单射的。

接下来看维数。 $\dim(\mathbf{P}_{n-m}(\mathbf{F}) \times \mathbf{P}_{m-1}(\mathbf{F})) = \dim \mathbf{P}_{n-m}(\mathbf{F}) + \dim \mathbf{P}_{m-1}(\mathbf{F}) = (n-m+1) + m = n+1$。 $\dim \mathbf{P}_n(\mathbf{F}) = n+1$。 $T$ 是一个从一个 $(n+1)$ 维空间到另一个 $(n+1)$ 维空间的单射线性映射。根据我们上一讲的结论，它必然是满射的。 因为 $T$ 是满射的，所以对任意 $p \in \mathbf{P}_n(\mathbf{F})$，都存在 $(q, r)$ 使得 $T(q, r) = p$。存在性得证。 因为 $T$ 是单射的，所以这样的 $(q, r)$ 是唯一的。唯一性得证。

4.C 多项式的零点

定义 4.9：多项式的零点 (zero of a polynomial) 称数 $\lambda \in \mathbf{F}$ 为多项式 $p \in \mathbf{P}(\mathbf{F})$ 的零点（或根），如果 $p(\lambda) = 0$。

定理 4.11：因子定理 (Factor Theorem) 多项式 $p \in \mathbf{P}(\mathbf{F})$ 的一个零点是 $\lambda \in \mathbf{F}$，当且仅当存在一个多项式 $q \in \mathbf{P}(\mathbf{F})$ 使得对每个 $z \in \mathbf{F}$ 均有 $$ p(z) = (z-\lambda)q(z) $$

证明： $(\Rightarrow)$ 假设 $\lambda$ 是 $p$ 的一个零点。 根据带余除法，存在多项式 $q$ 和 $r$ 使得 $p(z) = (z-\lambda)q(z) + r(z)$，且 $\text{deg } r < \text{deg}(z-\lambda) = 1$。 这意味着 $r(z)$ 是一个常数，记为 $c$。所以 $p(z) = (z-\lambda)q(z) + c$。 将 $z=\lambda$ 代入上式，得 $p(\lambda) = (\lambda-\lambda)q(\lambda) + c = c$。 因为 $\lambda$ 是零点，所以 $p(\lambda)=0$，因此 $c=0$。 所以 $p(z) = (z-\lambda)q(z)$。

$(\Leftarrow)$ 假设 $p(z) = (z-\lambda)q(z)$。将 $z=\lambda$ 代入，得 $p(\lambda) = (\lambda-\lambda)q(\lambda) = 0$。 所以 $\lambda$ 是 $p$ 的一个零点。证毕。

推论 4.12：多项式零点的个数 一个 $m$ 次多项式至多有 $m$ 个互不相同的零点。

证明： 我们对次数 $m$ 进行数学归纳。 基础情形： 当 $m=1$ 时，$p(z) = a_1 z + a_0$ ($a_1 \neq 0$)。它有唯一的零点 $z = -a_0/a_1$。结论成立。 归纳假设： 假设对任意 $m-1$ 次多项式，结论成立。 设 $p$ 是一个 $m$ 次多项式。如果 $p$ 没有零点，那么结论显然成立 (0 $\le m$)。 如果 $p$ 有一个零点 $\lambda$，根据因子定理，$p(z) = (z-\lambda)q(z)$，其中 $q$ 是一个 $m-1$ 次多项式。 $p$ 的任何其他零点都必须是 $q$ 的零点。 根据归纳假设，$q$ 至多有 $m-1$ 个互不相同的零点。 因此，$p$ 至多有 $1 + (m-1) = m$ 个互不相同的零点。证毕。

4.D 多项式的分解

代数学基本定理 这个定理是复分析中的一个深刻结果，我们在这里直接引用而不加证明。

定理 4.13：代数学基本定理 (Fundamental Theorem of Algebra) 每个次数大于等于1的复系数多项式都至少有一个复数零点。

这个定理保证了我们在复数域 $\mathbf{C}$ 中总能找到多项式的根。利用这个定理，我们可以证明任何复系数多项式都可以完全分解为一次因子的乘积。

推论 4.14：复系数多项式的分解 若 $p \in \mathbf{P}(\mathbf{C})$ 是一个 $m$ 次多项式 ($m \ge 1$)，则 $p$ 可以唯一地 (除因子次序外) 分解为 $$ p(z) = c(z-\lambda_1)\cdots(z-\lambda_m) $$ 其中 $c, \lambda_1, \ldots, \lambda_m \in \mathbf{C}$。

证明： 对次数 $m$ 进行归纳。 基础情形: $m=1$ 时，$p(z)=cz+d = c(z-(-d/c))$，成立。 归纳假设: 假设对任意 $m-1$ 次多项式结论成立。 设 $p$ 是一个 $m$ 次多项式。根据代数学基本定理，$p$ 至少有一个零点 $\lambda_1$。 根据因子定理，$p(z)=(z-\lambda_1)q(z)$，其中 $q$ 是一个 $m-1$ 次多项式。 根据归纳假设，$q(z)$ 可以分解为 $c(z-\lambda_2)\cdots(z-\lambda_m)$。 代入即得 $p(z)$ 的分解形式。唯一性源于零点集合的唯一性。证毕。

实系数多项式的分解 实系数多项式在实数域 $\mathbf{R}$ 中不一定有零点（例如 $x^2+1$）。但如果把它看作复系数多项式，它的复数零点具有一个很好的性质。

命题 4.15：实系数多项式的共轭零点 设 $p$ 是一个实系数多项式。如果 $\lambda \in \mathbf{C}$ 是 $p$ 的一个零点，则其共轭 $\bar{\lambda}$ 也是 $p$ 的一个零点。

证明： 设 $p(z) = a_0 + a_1 z + \cdots + a_m z^m$，其中所有 $a_j \in \mathbf{R}$。 假设 $p(\lambda) = 0$，即 $a_0 + a_1 \lambda + \cdots + a_m \lambda^m = 0$。 对整个等式取共轭： $$ \overline{a_0 + a_1 \lambda + \cdots + a_m \lambda^m} = \overline{0} $$ 利用共轭的性质： $$ \overline{a_0} + \overline{a_1 \lambda} + \cdots + \overline{a_m \lambda^m} = 0 $$ $$ \overline{a_0} + \overline{a_1}\bar{\lambda} + \cdots + \overline{a_m}\overline{\lambda^m} = 0 $$ 因为 $a_j$ 是实数，所以 $\overline{a_j} = a_j$。因此： $$ a_0 + a_1 \bar{\lambda} + \cdots + a_m (\bar{\lambda})^m = 0 $$ 这正是 $p(\bar{\lambda}) = 0$ 的定义。证毕。

利用这个性质，我们可以得到实系数多项式的分解定理。

定理 4.17：实系数多项式的分解 若 $p \in \mathbf{P}(\mathbf{R})$ 是一个非常数多项式，则 $p$ 可以分解为如下形式的乘积： $$ p(x) = c(x-\lambda_1)\cdots(x-\lambda_k)(x^2+b_1x+c_1)\cdots(x^2+b_Mx+c_M) $$ 其中 $c, \lambda_j, b_j, c_j$ 都是实数，并且对每个 $j$ 都有 $b_j^2 < 4c_j$。

证明： 将 $p$ 视为一个复系数多项式。根据推论 4.14，它可以完全分解为 $p(x) = c(x-z_1)\cdots(x-z_m)$。 $p$ 的非实数零点必然成共轭对出现。我们将每一对共轭零点 $(\lambda, \bar{\lambda})$ 配对。对应的因子是 $(x-\lambda)(x-\bar{\lambda})$。 $$ \begin{aligned} (x-\lambda)(x-\bar{\lambda}) &= x^2 - (\lambda+\bar{\lambda})x + \lambda\bar{\lambda} \\ &= x^2 - (2\text{Re }\lambda)x + |\lambda|^2 \end{aligned} $$ 令 $b = -2\text{Re }\lambda$ 和 $c = |\lambda|^2$。这两个都是实数。所以我们得到一个二次因子 $x^2+bx+c$。这个二次多项式的判别式是 $b^2 - 4c = ( -2\text{Re }\lambda)^2 - 4|\lambda|^2 = 4(\text{Re }\lambda)^2 - 4((\text{Re }\lambda)^2+(\text{Im }\lambda)^2) = -4(\text{Im }\lambda)^2$。因为 $\lambda$ 是非实数零点，所以 $\text{Im }\lambda \neq 0$，因此判别式小于0。 将所有非实数零点配对，剩下的零点都是实数。将它们组合起来，就得到了定理中的分解形式。证毕。

第四讲小结

今天我们集中讨论了多项式的一些关键性质，这些性质是我们后续学习的基石。

  1. 我们回顾了复数的基本运算，特别是共轭和绝对值。
  2. 我们严格证明了多项式的系数是唯一的，并学习了带余除法定理。
  3. 我们探讨了多项式的零点，证明了因子定理，并得出一个 $m$ 次多项式至多有 $m$ 个零点的结论。
  4. 最后，我们学习了最重要的分解定理：基于代数学基本定理，任何复系数多项式都可以分解为一次因子的乘积；而任何实系数多项式都可以分解为一次和二次不可约因子的乘积。
