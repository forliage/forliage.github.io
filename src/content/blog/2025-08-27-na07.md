# 数值分析07:牛顿插值法与差商

### **引言：寻求更高效的插值方案**

在上一讲中，我们学习了拉格朗日多项式插值。它从理论上完美地解决了“给定 $n+1$ 个点，如何构造一个次数不超过 $n$ 的多项式穿过它们”的问题。拉格朗日的方法通过构造一组巧妙的基函数 $L\_{n,i}(x)$，给出了插值多项式 $P\_n(x) = \\sum\\limits\_{i=0}^n y\_i L\_{n,i}(x)$ 一个清晰的、具有存在性和唯一性证明的表达形式。

然而，从**计算实践**的角度来看，拉格朗日插值法存在一个显著的**缺陷**： **缺乏可扩展性 (Lack of Recursiveness)。** 想象一下，我们已经辛苦地基于 $n+1$ 个数据点 $(x\_0, y\_0), \\dots, (x\_n, y\_n)$ 构造了 $n$ 次插值多项式 $P\_n(x)$。现在，我们获得了一个新的数据点 $(x\_{n+1}, y\_{n+1})$。如果我们想得到一个能穿过所有 $n+2$ 个点的、新的 $n+1$ 次插值多项式 $P\_{n+1}(x)$，我们该怎么做？

使用拉格朗日方法，我们几乎必须**推倒重来**。因为新的基函数 $L\_{n+1, i}(x)$ 与旧的 $L\_{n,i}(x)$ 完全不同，所有的计算都需要重新进行。这在需要逐步增加数据点来观察插值效果变化的场景中，效率极低。

这就引出了我们今天的核心问题：**是否存在一种插值多项式的表示形式，使得我们可以在旧的 $n$ 次多项式的基础上，通过添加一个“修正项”来方便地得到新的 $n+1$ 次多项式？**

答案是肯定的。这就是**牛顿插值法**的精髓。它将插值多项式表示为一种嵌套的、可扩展的形式，而实现这一切的数学工具，就是**差商 (Divided Differences)**。

今天，我们将学习：

1.  **拉格朗日插值的实践问题：** 通过一个例子直观感受插值与外推的差异，以及高次插值的潜在风险。
2.  **内维尔方法 (Neville's Method):** 一种优雅的、生成特定点插值的递推算法，为牛顿法的思想做铺垫。
3.  **差商：** 定义和计算这个核心概念，它类似于离散世界中的“导数”。
4.  **牛顿插值多项式：** 如何利用差商来构建具有完美扩展性的插值多项式。
5.  **等距节点的特殊情况：** 介绍牛顿前向/后向差分公式，简化计算。

### **第一部分：拉格朗日插值的实践与思考**

#### **1.1 案例分析：插值 vs. 外推**

我们通过一个例子来具体看下拉格朗日插值的应用。 **例：** 已知 $\\sin(\\pi/6)=0.5$, $\\sin(\\pi/4) \\approx 0.7071$, $\\sin(\\pi/3) \\approx 0.8660$。使用线性和二次拉格朗日插值来近似计算 $\\sin(50^\\circ)$ 的值。($50^\\circ = 5\\pi/18$ 弧度)。 已知真值 $\\sin(50^\\circ) \\approx 0.7660444$。

**a) 线性插值 (Linear Interpolation)**

*   **Case 1 (外推 Extrapolation):** 使用节点 $x\_0 = \\pi/6$ (30°) 和 $x\_1 = \\pi/4$ (45°)。目标点 $x=50^\\circ$ 在插值区间之外。 $$P\_1(x) = y\_0 \\frac{x-x\_1}{x\_0-x\_1} + y\_1 \\frac{x-x\_0}{x\_1-x\_0}$$ $$P\_1(5\\pi/18) \\approx 0.7761$$ **误差:** $|0.7761 - 0.7660| \\approx 0.0101$ (相对较大)。
*   **Case 2 (内插 Interpolation):** 使用节点 $x\_1 = \\pi/4$ (45°) 和 $x\_2 = \\pi/3$ (60°)。目标点 $x=50^\\circ$ 在插值区间之内。 $$P\_1(x) = y\_1 \\frac{x-x\_2}{x\_1-x\_2} + y\_2 \\frac{x-x\_1}{x\_2-x\_1}$$ $$P\_1(5\\pi/18) \\approx 0.76008$$ **误差:** $|0.76008 - 0.7660| \\approx 0.0059$ (小得多)。

**结论：** "In general, interpolation is better than extrapolation."外推的误差通常比内插大得多，因为我们对函数在插值区间外的行为一无所知，外推本质上是基于一种“趋势会延续”的盲目假设。

**b) 二次插值 (Quadratic Interpolation)** 使用所有三个节点 $x\_0 = \\pi/6, x\_1 = \\pi/4, x\_2 = \\pi/3$。目标点 $x=50^\\circ$ 位于插值区间内。 $$P\_2(x) = y\_0 L\_{2,0}(x) + y\_1 L\_{2,1}(x) + y\_2 L\_{2,2}(x)$$ 计算可得 $P\_2(5\\pi/18) \\approx 0.76543$。 **误差:** $|0.76543 - 0.7660| \\approx 0.00061$。

**结论：** "Interpolation with higher degree usually gives better results."在这个例子中，二次插值的误差比线性插值小了一个数量级。

#### **1.2 高次插值的陷阱：龙格现象 (Runge's Phenomenon)**

"The higher the better? Nooooooo...." 增加插值多项式的次数并不总是能提高精度。对于某些函数，当使用高次多项式在等距节点上进行插值时，可能会在区间的边缘出现剧烈的、非物理的振荡，导致巨大的误差。这就是著名的**龙格现象**。

**经典例子：** 对函数 $f(x) = \\frac{1}{1+25x^2}$ 在区间 $\[-1, 1\]$ 上使用 $n+1$ 个等距节点进行插值。

尽管插值多项式在中间部分拟合得不错，但在靠近-1和1的区域，它剧烈振荡，与原函数偏差极大。而且，随着次数 $n$ 的增加，这种振荡会越来越严重。

**教训：**

1.  盲目使用高次多项式插值是危险的。
2.  插值节点的**选择**至关重要。（使用切比雪夫节点可以极大地缓解龙格现象）。
3.  对于大量数据点，使用分段低次插值（如样条插值）通常是更好的选择。

### **第二部分：牛顿插值法与差商**

"Excellent point! Let's look at Neville's Method..."在直接进入牛顿法之前，内维尔方法为我们展示了插值问题中优雅的递推结构。

#### **3.1.1 内维尔方法 (Neville's Method)**

**目标：** 给定 $n+1$ 个点，高效地计算出插值多项式在**某一个特定点 $x$** 的值 $P\_n(x)$，而不需要显式地写出 $P\_n(x)$ 的表达式。

**记号：** $P\_{m\_1, \\dots, m\_k}(x)$ 表示使用节点 $(x\_{m\_1}, y\_{m\_1}), \\dots, (x\_{m\_k}, y\_{m\_k})$ 构造的插值多项式在点 $x$ 的值。

**递推关系：** 可以证明，一个更高阶的插值多项式可以由两个低一阶的插值多项式线性组合而成： $$P\_{i, i+1, \\dots, j}(x) = \\frac{(x-x\_i)P\_{i+1, \\dots, j}(x) - (x-x\_j)P\_{i, \\dots, j-1}(x)}{x\_j - x\_i}$$

**计算过程：** 我们可以构造一个表格。 第0列是已知的函数值 $P\_i = y\_i$。 第1列由第0列计算得到：$P\_{i, i+1}$ 第2列由第1列计算得到：$P\_{i, i+1, i+2}$ ... 最终表格右上角的元素 $P\_{0,1,\\dots,n}$ 就是我们要求的值。

**示例：** $x\_0 \\quad P\_0$ $x\_1 \\quad P\_1 \\quad P\_{0,1}$ $x\_2 \\quad P\_2 \\quad P\_{1,2} \\quad P\_{0,1,2}$ $x\_3 \\quad P\_3 \\quad P\_{2,3} \\quad P\_{1,2,3} \\quad P\_{0,1,2,3}$

内维尔方法非常适合于“试探性”的插值，即逐步增加节点数量，观察插值结果的变化趋势。但它的主要目的是求值，而不是得到多项式本身。

#### **3.2 差商 (Divided Differences)**

牛顿插值法的核心是**差商**。差商是多项式系数的“天然”表示，并且它与函数导数有着深刻的联系。

**定义：**

*   **零阶差商 (0-th DD):** $f\[x\_i\] = f(x\_i) = y\_i$
    
*   **一阶差商 (1-st DD):** $f\[x\_i, x\_j\] = \\frac{f\[x\_j\] - f\[x\_i\]}{x\_j - x\_i}$
    
    *   这正是连接 $(x\_i, y\_i)$ 和 $(x\_j, y\_j)$ 两点直线的**斜率**。
*   **二阶差商 (2-nd DD):** $f\[x\_i, x\_j, x\_k\] = \\frac{f\[x\_j, x\_k\] - f\[x\_i, x\_j\]}{x\_k - x\_i}$
    
    *   这可以看作是“斜率的斜率”，类似于二阶导数。
*   **k阶差商 (k-th DD):** $$f\[x\_0, x\_1, \\dots, x\_k\] = \\frac{f\[x\_1, \\dots, x\_k\] - f\[x\_0, \\dots, x\_{k-1}\]}{x\_k - x\_0}$$
    

**重要性质：**

*   **对称性：** "The value of $f\[x\_0, \\dots, x\_k\]$ is independent of the order of the numbers $x\_0, \\dots, x\_k$." 这个性质非常重要，但从递推定义中不易看出。它保证了我们选择节点的顺序不影响最终结果。
*   **与导数的关系：** 如果函数 $f$ 足够光滑，那么存在一点 $\\xi$ 在节点区间内，使得： $f\[x\_0, x\_1, \\dots, x\_k\] = \\frac{f^{(k)}(\\xi)}{k!}$ 这表明，$k$ 阶差商是 $k$ 阶导数除以 $k!$ 的一个离散模拟。

**差商表 (Divided Difference Table):** 我们可以像内维尔方法一样，用表格来系统地计算所有需要的差商。 $x\_0 \\quad f\[x\_0\]$ $\\qquad \\qquad f\[x\_0, x\_1\]$ $x\_1 \\quad f\[x\_1\] \\qquad \\qquad f\[x\_0, x\_1, x\_2\]$ $\\qquad \\qquad f\[x\_1, x\_2\] \\qquad \\qquad \\quad \\ddots$ $x\_2 \\quad f\[x\_2\] \\qquad \\qquad f\[x\_1, x\_2, x\_3\]$ $\\qquad \\qquad f\[x\_2, x\_3\]$ $x\_3 \\quad f\[x\_3\]$

#### **3.2.1 牛顿插值多项式**

牛顿插值法的思想是构造一个具有“可加性”的多项式。 $$P\_n(x) = P\_{n-1}(x) + (\\text{a correction term})$$ 这个修正项必须满足：

1.  它不改变 $P\_n(x)$ 在旧节点 $x\_0, \\dots, x\_{n-1}$ 上的值（即它在这些点上必须为0）。
2.  它保证 $P\_n(x\_n) = y\_n$。

满足条件1的最简单形式是 $C \\cdot (x-x\_0)(x-x\_1)\\dots(x-x\_{n-1})$。 由此，我们可以将插值多项式写成如下**牛顿形式 (Newton Form)**：

$$N\_n(x) = a\_0 + a\_1(x-x\_0) + a\_2(x-x\_0)(x-x\_1) + \\dots + a\_n(x-x\_0)\\dots(x-x\_{n-1})$$

**如何确定系数 $a\_k$？** 通过依次代入插值条件：

*   $x=x\_0: N\_n(x\_0) = a\_0 = y\_0 = f\[x\_0\]$
*   $x=x\_1: N\_n(x\_1) = a\_0 + a\_1(x\_1-x\_0) = y\_1 \\implies a\_1 = \\frac{y\_1-y\_0}{x\_1-x\_0} = f\[x\_0, x\_1\]$
*   $x=x\_2: \\dots \\implies a\_2 = f\[x\_0, x\_1, x\_2\]$
*   ...
*   **一般地，系数 $a\_k$ 就是 $k$ 阶差商：$a\_k = f\[x\_0, x\_1, \\dots, x\_k\]$**

**牛顿插值多项式公式：** $$N\_n(x) = f\[x\_0\] + \\sum\_{k=1}^n f\[x\_0, \\dots, x\_k\] \\prod\_{i=0}^{k-1} (x-x\_i)$$

**计算优势：**

*   **可扩展性：** 如果要从 $N\_n(x)$ 得到 $N\_{n+1}(x)$，我们只需要计算新的一行差商，然后在 $N\_n(x)$ 的表达式后面**加上一项** $f\[x\_0, \\dots, x\_{n+1}\](x-x\_0)\\dots(x-x\_n)$ 即可。之前的计算完全可以复用。
*   **高效求值：** 牛顿形式的多项式可以用嵌套乘法（秦九韶算法的变体）来高效求值。 $N\_n(x) = f\[x\_0\] + (x-x\_0)(f\[x\_0,x\_1\] + (x-x\_1)(f\[x\_0,x\_1,x\_2\] + \\dots))$

**误差项：** 由于插值多项式是唯一的，$N\_n(x) \\equiv P\_n(x)$，它们有完全相同的误差项： $$f(x) - N\_n(x) = f\[x\_0, \\dots, x\_n, x\] \\prod\_{i=0}^n (x-x\_i)$$

#### **3.2.2 等距节点：牛顿差分公式**

当插值节点是等距的，$x\_i = x\_0 + i h$，差商的计算可以大大简化，并用**差分 (Finite Differences)** 来表示。

**前向差分 (Forward Differences):** $\\Delta$ $$\\Delta f\_i = f\_{i+1} - f\_i$$ $$\\Delta^k f\_i = \\Delta(\\Delta^{k-1} f\_i) = \\Delta^{k-1} f\_{i+1} - \\Delta^{k-1} f\_i$$ 可以证明：$f\[x\_0, \\dots, x\_k\] = \\frac{\\Delta^k f\_0}{k! h^k}$。 令 $x=x\_0+sh$，则 $x-x\_i = (s-i)h$。代入牛顿公式得到：

**牛顿前向差分公式 (Newton's Forward-Difference Formula):** $$P\_n(x) = P\_n(x\_0+sh) = \\sum\_{k=0}^n \\binom{s}{k} \\Delta^k f\_0$$ 其中 $\\binom{s}{k} = \\frac{s(s-1)\\dots(s-k+1)}{k!}$ 是广义二项式系数。 这个公式最适合在数据表格的**开始部分**进行插值。

**后向差分 (Backward Differences):** $\\nabla$ $\\nabla f\_i = f\_i - f\_{i-1}$ 类似地，可以得到： **牛顿后向差分公式 (Newton's Backward-Difference Formula):** $$P\_n(x) = P\_n(x\_n+sh) = \\sum\_{k=0}^n (-1)^k \\binom{-s}{k} \\nabla^k f\_n$$ 这个公式使用表格末尾的差分值，最适合在数据表格的**结束部分**进行插值。

### **课程总结**

**本讲核心要点：**

1.  **高次多项式插值需谨慎。** 虽然增加次数在局部可以提高精度，但对于等距节点和某些函数，它可能导致灾难性的**龙格现象**。
2.  **牛顿插值法**通过**差商**的概念，构建了一种**可扩展**的插值多项式形式，克服了拉格朗日法计算效率低下的问题。
3.  **差商**可以看作是导数的离散模拟，其值与节点的排列顺序无关。
4.  计算差商并构建牛顿多项式，通常通过构造**差商表**来系统地完成。
5.  当节点**等距**时，差商可以用更简单的**前向/后向差分**代替，得到牛顿前向/后向差分插值公式，特别适用于处理表格数据。
