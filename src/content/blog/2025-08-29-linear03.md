# 线性代数03:线性映射

迄今为止我们所关注的都是向量空间，这并不怎么让人兴奋。线性代数真正让人感兴趣的部分是对要讨论的主题——线性映射的研究。

在本章中，我们将始终采用以下假定：

*   $\\mathbf{F}$ 表示 $\\mathbf{R}$ 或 $\\mathbf{C}$。
*   $V$ 和 $W$ 表示 $\\mathbf{F}$ 上的向量空间。

### **3.A 向量空间的线性映射**

#### **线性映射的定义和例子**

我们首先来定义线性代数中最重要的概念。

**定义 3.2：线性映射 (linear map)** 从 $V$ 到 $W$ 的一个**线性映射**是具有下列性质的函数 $T: V \\to W$：

1.  **加性 (additivity):** 对所有 $u, v \\in V$ 都有 $T(u+v) = Tu + Tv$。
2.  **齐性 (homogeneity):** 对所有 $\\lambda \\in \\mathbf{F}$ 和 $v \\in V$ 都有 $T(\\lambda v) = \\lambda(Tv)$。

有些数学家使用“线性变换”(linear transformation) 这个术语，意思与线性映射相同。

**记号 3.3：$\\mathcal{L}(V, W)$** 从 $V$ 到 $W$ 的所有线性映射构成的集合记为 $\\mathcal{L}(V, W)$。

**例子 3.4：线性映射的例子**

*   **零映射 (zero):** 定义 $0: V \\to W$ 为对所有 $v \\in V$ 都有 $0v = 0$。这是一个线性映射，即 $0 \\in \\mathcal{L}(V, W)$。
*   **恒等映射 (identity):** 定义 $I: V \\to V$ 为对所有 $v \\in V$ 都有 $Iv=v$。这是一个线性映射，即 $I \\in \\mathcal{L}(V, V)$。
*   **微分 (differentiation):** 定义 $D: \\mathbf{P}(\\mathbf{R}) \\to \\mathbf{P}(\\mathbf{R})$ 为 $Dp = p'$。这是一个线性映射。因为 $(p+q)' = p'+q'$ 和 $(\\lambda p)' = \\lambda p'$。
*   **积分 (integration):** 定义 $T: \\mathbf{P}(\\mathbf{R}) \\to \\mathbf{R}$ 为 $Tp = \\int\_0^1 p(x)dx$。这是一个线性映射。因为积分具有可加性和齐性。
*   **从 $\\mathbf{F}^n$ 到 $\\mathbf{F}^m$ 的映射:** 设 $A\_{j,k} \\in \\mathbf{F}$ 是一些常数。定义 $T: \\mathbf{F}^n \\to \\mathbf{F}^m$ 为： $$ T(x\_1, \\ldots, x\_n) = \\left( \\sum\_{k=1}^n A\_{1,k}x\_k, \\ldots, \\sum\_{k=1}^n A\_{m,k}x\_k \\right) $$ 这也是一个线性映射。

**例子：一个具体的线性映射** 判断函数 $T: \\mathbf{R}^3 \\to \\mathbf{R}^2$，$T(x,y,z) = (2x-y+3z, 7x+5y-6z)$ 是否为线性映射。

**解：** 我们需要验证加性和齐性。

1.  **加性:** 设 $u = (x\_1, y\_1, z\_1)$ 和 $v = (x\_2, y\_2, z\_2)$ 是 $\\mathbf{R}^3$ 中的任意两个向量。 $$ \\begin{aligned} T(u+v) &= T(x\_1+x\_2, y\_1+y\_2, z\_1+z\_2) \\\\ &= (2(x\_1+x\_2) - (y\_1+y\_2) + 3(z\_1+z\_2), 7(x\_1+x\_2) + 5(y\_1+y\_2) - 6(z\_1+z\_2)) \\\\ &= ((2x\_1-y\_1+3z\_1)+(2x\_2-y\_2+3z\_2), (7x\_1+5y\_1-6z\_1)+(7x\_2+5y\_2-6z\_2)) \\\\ &= (2x\_1-y\_1+3z\_1, 7x\_1+5y\_1-6z\_1) + (2x\_2-y\_2+3z\_2, 7x\_2+5y\_2-6z\_2) \\\\ &= Tu + Tv \\end{aligned} $$ 加性成立。
2.  **齐性:** 设 $\\lambda \\in \\mathbf{R}$ 和 $u=(x,y,z) \\in \\mathbf{R}^3$。 $$ \\begin{aligned} T(\\lambda u) &= T(\\lambda x, \\lambda y, \\lambda z) \\\\ &= (2(\\lambda x) - (\\lambda y) + 3(\\lambda z), 7(\\lambda x) + 5(\\lambda y) - 6(\\lambda z)) \\\\ &= (\\lambda(2x-y+3z), \\lambda(7x+5y-6z)) \\\\ &= \\lambda(2x-y+3z, 7x+5y-6z) \\\\ &= \\lambda(Tu) \\end{aligned} $$ 齐性成立。 因此，$T$ 是一个线性映射。

线性映射有一个非常重要的性质：只要我们知道了它在一个基上的取值，我们就能确定它在整个空间上的取值。

**命题 3.5：线性映射由其在基上的值确定** 设 $(v\_1, \\ldots, v\_n)$ 是 $V$ 的一个基，$(w\_1, \\ldots, w\_n)$ 是 $W$ 中的任意一组向量。则存在唯一的线性映射 $T: V \\to W$ 使得对每个 $j=1, \\ldots, n$ 都有 $Tv\_j = w\_j$。

**证明：** **存在性：** 我们来定义一个函数 $T: V \\to W$。因为 $(v\_1, \\ldots, v\_n)$ 是 $V$ 的基，所以 $V$ 中任意向量 $v$ 都可以唯一地表示为 $v = c\_1v\_1 + \\cdots + c\_n v\_n$。我们定义 $T$ 为： $$ Tv = c\_1w\_1 + \\cdots + c\_n w\_n $$ 显然，对于这个定义，有 $Tv\_j = w\_j$。现在我们证明 $T$ 是线性的。 设 $u = a\_1v\_1 + \\cdots + a\_n v\_n$，则 $u+v = (a\_1+c\_1)v\_1 + \\cdots + (a\_n+c\_n)v\_n$。根据 $T$ 的定义： $$ T(u+v) = (a\_1+c\_1)w\_1 + \\cdots + (a\_n+c\_n)w\_n = (a\_1w\_1+\\cdots+a\_nw\_n) + (c\_1w\_1+\\cdots+c\_nw\_n) = Tu+Tv $$ 类似地，对于 $\\lambda \\in \\mathbf{F}$，有 $\\lambda v = (\\lambda c\_1)v\_1 + \\cdots + (\\lambda c\_n)v\_n$。 $$ T(\\lambda v) = (\\lambda c\_1)w\_1 + \\cdots + (\\lambda c\_n)w\_n = \\lambda(c\_1w\_1 + \\cdots + c\_n w\_n) = \\lambda(Tv) $$ 所以 $T$ 是线性的。存在性得证。

**唯一性：** 假设 $S: V \\to W$ 是另一个满足 $Sv\_j = w\_j$ 的线性映射。对于任意 $v = c\_1v\_1 + \\cdots + c\_n v\_n \\in V$，我们有： $$ Sv = S(c\_1v\_1 + \\cdots + c\_n v\_n) = c\_1Sv\_1 + \\cdots + c\_n Sv\_n = c\_1w\_1 + \\cdots + c\_n w\_n = Tv $$ 因为 $S$ 和 $T$ 对 $V$ 中任意向量的作用都相同，所以 $S=T$。唯一性得证。

#### **$\\mathcal{L}(V, W)$ 上的代数运算**

$\\mathcal{L}(V, W)$ 本身也是一个向量空间。

**定义 3.6：$\\mathcal{L}(V, W)$ 上的加法和标量乘法** 设 $S, T \\in \\mathcal{L}(V, W)$ 和 $\\lambda \\in \\mathbf{F}$。

*   **和 $S+T$** 定义为：$(S+T)v = Sv + Tv$
*   **积 $\\lambda T$** 定义为：$(\\lambda T)v = \\lambda(Tv)$

可以验证，在这些运算下，$\\mathcal{L}(V, W)$ 是一个向量空间。

**定义 3.8：线性映射的乘积 (product of linear maps)** 设 $T \\in \\mathcal{L}(U, V)$ และ $S \\in \\mathcal{L}(V, W)$。乘积 $ST$ 定义为函数 $(ST)(u) = S(Tu)$，其中 $u \\in U$。 这个乘积就是函数的复合。可以证明 $ST \\in \\mathcal{L}(U, W)$。

### **3.B 零空间与值域**

对于每个线性映射，都有两个与之关联的非常重要的子空间。

**定义 3.12：零空间 (null space)** 一个线性映射 $T: V \\to W$ 的**零空间**，记为 $\\text{null } T$，是指 $V$ 中被 $T$ 映为 $0$ 的向量构成的子集： $$ \\text{null } T = {v \\in V : Tv = 0} $$ 零空间有时也称为**核 (kernel)**。

**例子 3.13：零空间的例子**

*   考虑微分映射 $D \\in \\mathcal{L}(\\mathbf{P}(\\mathbf{R}), \\mathbf{P}(\\mathbf{R}))$。$Dp=0$ 当且仅当 $p$ 是常数函数。所以 $\\text{null } D$ 是所有常数函数构成的集合。
*   考虑映射 $T: \\mathbf{F}^3 \\to \\mathbf{F}$，$T(z\_1, z\_2, z\_3) = z\_1+2z\_2+3z\_3$。则 $\\text{null } T = {(z\_1, z\_2, z\_3) \\in \\mathbf{F}^3 : z\_1+2z\_2+3z\_3 = 0}$。这是 $\\mathbf{F}^3$ 中过原点的一个平面。

**命题 3.14：零空间是子空间** 设 $T \\in \\mathcal{L}(V, W)$。则 $\\text{null } T$ 是 $V$ 的一个子空间。

**证明：** 我们需要验证三个条件：

1.  **加法单位元:** $T(0)=0$ (由 $T(0)=T(0+0)=T(0)+T(0)$ 可得)，所以 $0 \\in \\text{null } T$。
2.  **加法封闭性:** 设 $u, v \\in \\text{null } T$。则 $Tu=0, Tv=0$。于是 $T(u+v) = Tu + Tv = 0+0 = 0$。所以 $u+v \\in \\text{null } T$。
3.  **标量乘法封闭性:** 设 $\\lambda \\in \\mathbf{F}$ 且 $u \\in \\text{null } T$。则 $Tu=0$。于是 $T(\\lambda u) = \\lambda (Tu) = \\lambda(0) = 0$。所以 $\\lambda u \\in \\text{null } T$。 因此，$\\text{null } T$ 是 $V$ 的一个子空间。证毕。

零空间可以用来判断一个线性映射是否为单射。

**定义 3.15：单射 (injective)** 一个函数 $T: V \\to W$ 称为**单射**，如果当 $u, v \\in V$ 且 $u \\neq v$ 时，有 $Tu \\neq Tv$。

**命题 3.16：单射性等价于零空间为${0}$** 设 $T \\in \\mathcal{L}(V, W)$。则 $T$ 是单射的当且仅当 $\\text{null } T = {0}$。

**证明：** $(\\Rightarrow)$ 假设 $T$ 是单射的。我们知道 $T(0)=0$，所以 ${0} \\subseteq \\text{null } T$。为了证明反向包含，设 $v \\in \\text{null } T$。则 $Tv = 0 = T(0)$。因为 $T$ 是单射的，所以必有 $v=0$。因此 $\\text{null } T \\subseteq {0}$。所以 $\\text{null } T = {0}$。

$(\\Leftarrow)$ 假设 $\\text{null } T = {0}$。设 $u, v \\in V$ 且 $Tu=Tv$。则 $T(u-v) = Tu-Tv = 0$。所以 $u-v \\in \\text{null } T$。因为 $\\text{null } T = {0}$，所以 $u-v=0$，即 $u=v$。因此 $T$ 是单射的。证毕。

**定义 3.17：值域 (range)** 一个函数 $T: V \\to W$ 的**值域**，记为 $\\text{range } T$，是 $W$ 中形如 $Tv$（其中 $v \\in V$）的向量构成的子集： $$ \\text{range } T = {Tv : v \\in V} $$

**命题 3.19：值域是子空间** 设 $T \\in \\mathcal{L}(V, W)$。则 $\\text{range } T$ 是 $W$ 的一个子空间。

**证明：**

1.  **加法单位元:** $T(0)=0$，所以 $0 \\in \\text{range } T$。
2.  **加法封闭性:** 设 $w\_1, w\_2 \\in \\text{range } T$。则存在 $v\_1, v\_2 \\in V$ 使得 $Tv\_1=w\_1, Tv\_2=w\_2$。于是 $w\_1+w\_2 = Tv\_1+Tv\_2 = T(v\_1+v\_2)$。因为 $v\_1+v\_2 \\in V$，所以 $w\_1+w\_2 \\in \\text{range } T$。
3.  **标量乘法封闭性:** 设 $\\lambda \\in \\mathbf{F}$ 且 $w \\in \\text{range } T$。则存在 $v \\in V$ 使得 $Tv=w$。于是 $\\lambda w = \\lambda(Tv) = T(\\lambda v)$。因为 $\\lambda v \\in V$，所以 $\\lambda w \\in \\text{range } T$。 因此，$\\text{range } T$ 是 $W$ 的一个子空间。证毕。

**定义 3.20：满射 (surjective)** 一个函数 $T: V \\to W$ 称为**满射**，如果它的值域等于 $W$，即 $\\text{range } T = W$。

现在我们来到本章的核心定理，它揭示了零空间和值域维数之间的深刻联系。

**定理 3.22：线性映射基本定理 (Fundamental Theorem of Linear Maps)** 设 $V$ 是有限维的，且 $T \\in \\mathcal{L}(V, W)$。则 $\\text{range } T$ 是有限维的，并且 $$ \\dim V = \\dim(\\text{null } T) + \\dim(\\text{range } T) $$ 这个定理也常被称为**秩-零化度定理 (Rank-Nullity Theorem)**。

**证明：** 设 $(u\_1, \\ldots, u\_m)$ 是 $\\text{null } T$ 的一个基。则 $\\dim(\\text{null } T) = m$。这是一个在 $V$ 中的线性无关组，我们可以将其扩充为 $V$ 的一个基：$(u\_1, \\ldots, u\_m, v\_1, \\ldots, v\_n)$。于是 $\\dim V = m+n$。 为了完成证明，我们只需证明 $(Tv\_1, \\ldots, Tv\_n)$ 是 $\\text{range } T$ 的一个基。这样就有 $\\dim(\\text{range } T) = n$，定理即得证。

**证明 $(Tv\_1, \\ldots, Tv\_n)$ 张成 $\\text{range } T$:** 任取 $v \\in V$。因为 $(u\_1, \\ldots, u\_m, v\_1, \\ldots, v\_n)$ 是 $V$ 的基，所以 $$ v = a\_1u\_1 + \\cdots + a\_m u\_m + b\_1v\_1 + \\cdots + b\_n v\_n $$ 对上式两边作用 $T$ 得： $$ \\begin{aligned} Tv &= T(a\_1u\_1 + \\cdots + a\_m u\_m + b\_1v\_1 + \\cdots + b\_n v\_n) \\\\ &= a\_1Tu\_1 + \\cdots + a\_m Tu\_m + b\_1Tv\_1 + \\cdots + b\_n Tv\_n \\\\ &= b\_1Tv\_1 + \\cdots + b\_n Tv\_n \\end{aligned} $$ (因为 $u\_j \\in \\text{null } T$，所以 $Tu\_j = 0$) 这表明 $\\text{range } T$ 中的任意向量都可以写成 $(Tv\_1, \\ldots, Tv\_n)$ 的线性组合。所以它们张成 $\\text{range } T$。

**证明 $(Tv\_1, \\ldots, Tv\_n)$ 线性无关:** 假设 $$ c\_1Tv\_1 + \\cdots + c\_n Tv\_n = 0 $$ 利用 $T$ 的线性性，我们得到 $$ T(c\_1v\_1 + \\cdots + c\_n v\_n) = 0 $$ 这说明向量 $c\_1v\_1 + \\cdots + c\_n v\_n$ 属于 $\\text{null } T$。因为 $(u\_1, \\ldots, u\_m)$ 是 $\\text{null } T$ 的基，所以存在标量 $d\_1, \\ldots, d\_m$ 使得 $$ c\_1v\_1 + \\cdots + c\_n v\_n = d\_1u\_1 + \\cdots + d\_m u\_m $$ 移项得： $$ c\_1v\_1 + \\cdots + c\_n v\_n - d\_1u\_1 - \\cdots - d\_m u\_m = 0 $$ 但是 $(u\_1, \\ldots, u\_m, v\_1, \\ldots, v\_n)$ 是 $V$ 的基，是线性无关的。因此，所有的系数都必须为0。特别地，$c\_1=\\cdots=c\_n=0$。 所以 $(Tv\_1, \\ldots, Tv\_n)$ 是线性无关的。 证毕。

### **3.C 矩阵**

我们现在将线性映射与矩阵联系起来。矩阵为线性映射提供了一种具体的计算工具。

**定义 3.32：线性映射的矩阵 (matrix of a linear map)** 设 $T \\in \\mathcal{L}(V, W)$，$(v\_1, \\ldots, v\_n)$ 是 $V$ 的基，$(w\_1, \\ldots, w\_m)$ 是 $W$ 的基。$T$ 关于这些基的**矩阵**，记为 $\\mathcal{M}(T)$，是一个 $m \\times n$ 矩阵，其第 $k$ 列的元素由 $Tv\_k$ 在基 $(w\_1, \\ldots, w\_m)$下的坐标确定。 也就是说，如果 $$ Tv\_k = A\_{1,k}w\_1 + \\cdots + A\_{m,k}w\_m $$ 则 $\\mathcal{M}(T)$ 的第 $j$ 行第 $k$ 列的元素是 $A\_{j,k}$。

**例子 3.33：矩阵的计算** 设 $T \\in \\mathcal{L}(\\mathbf{F}^2, \\mathbf{F}^3)$ 定义为 $T(x,y) = (x+3y, 2x+5y, 7x+9y)$。 求 $T$ 关于 $\\mathbf{F}^2$ 和 $\\mathbf{F}^3$ 的标准基的矩阵。

**解：** $\\mathbf{F}^2$ 的标准基是 $((1,0), (0,1))$。 $\\mathbf{F}^3$ 的标准基是 $((1,0,0), (0,1,0), (0,0,1))$。

我们需要计算 $T$ 作用在 $\\mathbf{F}^2$ 基向量上的结果，并用 $\\mathbf{F}^3$ 的基来表示： $$ T(1,0) = (1, 2, 7) = 1(1,0,0) + 2(0,1,0) + 7(0,0,1) $$ 所以矩阵的第一列是 $\\begin{pmatrix} 1 \\\\ 2 \\\\ 7 \\end{pmatrix}$。

$$ T(0,1) = (3, 5, 9) = 3(1,0,0) + 5(0,1,0) + 9(0,0,1) $$ 所以矩阵的第二列是 $\\begin{pmatrix} 3 \\\\ 5 \\\\ 9 \\end{pmatrix}$。

因此，$T$ 的矩阵是： $$ \\mathcal{M}(T) = \\begin{pmatrix} 1 & 3 \\\\ 2 & 5 \\\\ 7 & 9 \\end{pmatrix} $$

线性映射的加法和标量乘法，对应着矩阵的加法和标量乘法。更重要的是，线性映射的复合对应着矩阵的乘法。

**定义 3.41：矩阵乘法 (matrix multiplication)** 设 $A$ 是一个 $m \\times n$ 矩阵，$C$ 是一个 $n \\times p$ 矩阵。乘积 $AC$ 是一个 $m \\times p$ 矩阵，其第 $j$ 行第 $k$ 列的元素为： $$ (AC)\_{j,k} = \\sum\_{r=1}^n A\_{j,r}C\_{r,k} $$

**命题 3.43：线性映射乘积的矩阵** 设 $T \\in \\mathcal{L}(U, V)$ 和 $S \\in \\mathcal{L}(V, W)$。则 $$ \\mathcal{M}(ST) = \\mathcal{M}(S)\\mathcal{M}(T) $$

### **3.D 可逆性与同构的向量空间**

**定义 3.53：可逆的 (invertible)** 一个线性映射 $T \\in \\mathcal{L}(V, W)$ 称为**可逆的**，如果存在一个线性映射 $S \\in \\mathcal{L}(W, V)$ 使得 $ST = I\_V$ 且 $TS = I\_W$。 这个 $S$ 称为 $T$ 的**逆 (inverse)**，记为 $T^{-1}$。

**命题 3.56：可逆性等价于单射且满射** 一个线性映射是可逆的，当且仅当它既是单射的又是满射的。

**证明：** $(\\Rightarrow)$ 假设 $T$ 可逆，逆为 $T^{-1}$。

*   **单射性：** 若 $Tu=Tv$，则 $T^{-1}(Tu) = T^{-1}(Tv)$，即 $(T^{-1}T)u = (T^{-1}T)v$，所以 $u=v$。故 $T$ 是单射的。
*   **满射性：** 对任意 $w \\in W$，我们有 $w = I\_W w = (TT^{-1})w = T(T^{-1}w)$。这说明 $w$ 在 $T$ 的值域中。故 $T$ 是满射的。

$(\\Leftarrow)$ 假设 $T$ 是单射且满射的。 对每个 $w \\in W$，因为 $T$ 满射，所以存在 $v \\in V$ 使得 $Tv=w$。因为 $T$ 单射，这个 $v$ 是唯一的。我们定义一个函数 $S: W \\to V$ 为 $Sw = v$。 可以验证 $S$ 是线性的，并且满足 $ST=I\_V, TS=I\_W$。证毕。

**定义 3.58：同构 (isomorphism)**

*   一个可逆的线性映射称为**同构**。
*   如果存在从 $V$ 到 $W$ 的同构，则称向量空间 $V$ 与 $W$ 是**同构的**。

同构的向量空间在所有线性代数的性质上都是完全一样的，只是元素的“名字”不同。

**定理 3.59：维数完全刻画了同构** 两个有限维向量空间是同构的，当且仅当它们有相同的维数。

**证明：** $(\\Rightarrow)$ 设 $T \\in \\mathcal{L}(V, W)$ 是一个同构。因为 $T$ 是单射的，所以 $\\text{null } T = {0}$，即 $\\dim(\\text{null } T)=0$。因为 $T$ 是满射的，所以 $\\text{range } T = W$，即 $\\dim(\\text{range } T) = \\dim W$。 根据线性映射基本定理： $$ \\dim V = \\dim(\\text{null } T) + \\dim(\\text{range } T) = 0 + \\dim W = \\dim W $$

$(\\Leftarrow)$ 假设 $\\dim V = \\dim W = n$。 设 $(v\_1, \\ldots, v\_n)$ 是 $V$ 的一个基，$(w\_1, \\ldots, w\_n)$ 是 $W$ 的一个基。 根据命题 3.5，存在唯一的线性映射 $T: V \\to W$ 使得 $Tv\_j = w\_j$。

*   **满射性：** $\\text{range } T$ 包含 $(w\_1, \\ldots, w\_n)$，而后者张成 $W$，所以 $\\text{range } T = W$。$T$ 是满射的。
*   **单射性：** 因为 $T$ 满射，$\\dim(\\text{range } T) = n$。根据基本定理，$\\dim(\\text{null } T) = \\dim V - \\dim(\\text{range } T) = n - n = 0$。所以 $\\text{null } T = {0}$。$T$ 是单射的。 因为 $T$ 既是单射又是满射，所以它是同构。证毕。

### **3.E 向量空间的积与商**

#### **向量空间的积**

**定义 3.71: 向量空间的积 (product of vector spaces)** 设 $V\_1, \\ldots, V\_m$ 是向量空间。它们的**积** $V\_1 \\times \\cdots \\times V\_m$ 定义为集合 ${(v\_1, \\ldots, v\_m) : v\_j \\in V\_j}$，其上的运算为：

*   **加法:** $(u\_1, \\ldots, u\_m) + (v\_1, \\ldots, v\_m) = (u\_1+v\_1, \\ldots, u\_m+v\_m)$
*   **标量乘法:** $\\lambda(v\_1, \\ldots, v\_m) = (\\lambda v\_1, \\ldots, \\lambda v\_m)$

**命题 3.76：积的维数等于维数的和** 如果 $V\_1, \\ldots, V\_m$ 都是有限维的，则 $$ \\dim(V\_1 \\times \\cdots \\times V\_m) = \\dim V\_1 + \\cdots + \\dim V\_m $$ **证明:** 通过为每个 $V\_j$ 取一个基，然后将它们以自然的方式组合成积空间的基即可证明。

#### **商空间**

**定义 3.83: 商空间 (quotient space)** 设 $U$ 是 $V$ 的子空间。商空间 $V/U$ 是 $V$ 中所有 $U$ 的仿射子集（平行于 $U$）的集合，即 $$ V/U = {v+U : v \\in V} $$ 其上的运算定义为：

*   **加法:** $(v+U) + (w+U) = (v+w)+U$
*   **标量乘法:** $\\lambda(v+U) = (\\lambda v)+U$

**例子 3.84: 商空间的例子** 若 $V=\\mathbf{R}^2$，$U={(x, 2x) : x \\in \\mathbf{R}}$ (过原点斜率为2的直线)，则 $V/U$ 是 $\\mathbf{R}^2$ 中所有斜率为2的直线的集合。

**命题 3.89: 商空间的维数** 如果 $V$ 是有限维的，U 是 $V$ 的子空间，则 $$ \\dim(V/U) = \\dim V - \\dim U $$ **证明:** 考虑商映射 $\\pi: V \\to V/U$，定义为 $\\pi(v) = v+U$。 $\\pi$ 是一个线性映射。 $\\text{null } \\pi = {v \\in V : v+U = 0+U} = {v \\in V : v \\in U} = U$。 $\\text{range } \\pi = {v+U : v \\in V} = V/U$。 根据线性映射基本定理： $$ \\dim V = \\dim(\\text{null } \\pi) + \\dim(\\text{range } \\pi) = \\dim U + \\dim(V/U) $$ 移项即得证。

### **3.F 对偶**

#### **对偶空间与对偶基**

**定义 3.92: 线性泛函 (linear functional)** $V$ 上的一个**线性泛函**是从 $V$ 到 $\\mathbf{F}$ 的线性映射。即 $\\mathcal{L}(V, \\mathbf{F})$ 中的元素。

**定义 3.94: 对偶空间 (dual space)** $V$ 的**对偶空间**，记为 $V'$，是 $V$ 上所有线性泛函构成的向量空间，即 $V' = \\mathcal{L}(V, \\mathbf{F})$。

**命题 3.95: $\\dim V' = \\dim V$** 如果 $V$ 是有限维的，则 $V'$ 也是有限维的，且 $\\dim V' = \\dim V$。

**定义 3.96: 对偶基 (dual basis)** 设 $(v\_1, \\ldots, v\_n)$ 是 $V$ 的一个基。其**对偶基**是 $V'$ 中的一组向量 $(\\varphi\_1, \\ldots, \\varphi\_n)$，定义为： $$ \\varphi\_j(v\_k) = \\begin{cases} 1 & \\text{若 } k=j \\\\ 0 & \\text{若 } k \\neq j \\end{cases} $$

**命题 3.98：对偶基是基**

如果 $V$ 是有限维的，那么一个基的对偶基是其对偶空间的基。

**第三讲小结**

今天我们学习了线性代数的核心概念——线性映射。

1.  我们定义了什么是线性映射，并看了很多例子，理解了线性映射由其在基上的作用完全确定。
2.  我们研究了与每个线性映射关联的两个重要子空间：零空间和值域，并证明了它们的维数满足线性映射基本定理。
3.  我们学习了如何用矩阵来表示和计算线性映射，以及矩阵乘法如何对应于映射的复合。
4.  我们探讨了可逆性和同构的概念，并得到了一个深刻的结论：在有限维情况下，两个向量空间同构的充要条件是它们维数相同。
5.  最后，我们介绍了积空间、商空间和对偶空间这些更高级的构造，它们为我们提供了更强大的工具来分析向量空间的结构。
