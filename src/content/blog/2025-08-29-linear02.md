# 线性代数02:有限维向量空间

在本章中，我们将始终采用以下假定：

*   $\\mathbf{F}$ 表示 $\\mathbf{R}$ 或 $\\mathbf{C}$。
*   $V$ 表示 $\\mathbf{F}$ 上的向量空间。

### **2.A 张成空间与线性无关**

#### **线性组合与张成空间**

我们首先来定义如何用一组给定的向量来“生成”其他向量。

**记号 2.2：向量组 (list of vectors)** 我们用 $(v\_1, \\ldots, v\_m)$ 来表示 $V$ 中的一组向量。

**定义 2.3：线性组合 (linear combination)** $V$ 中一组向量 $(v\_1, \\ldots, v\_m)$ 的**线性组合**是指形如 $$ a\_1v\_1 + \\cdots + a\_m v\_m $$ 的向量，其中 $a\_1, \\ldots, a\_m \\in \\mathbf{F}$。

**例子 2.4：$\\mathbf{F}^3$ 中的线性组合**

在 $\\mathbf{F}^3$ 中，

*   向量 $(17, -4, 2)$ 是向量组 $((2, 1, -3), (1, -2, 4))$ 的线性组合，因为： $$ (17, -4, 2) = 6(2, 1, -3) + 5(1, -2, 4) $$
*   向量 $(17, -4, 5)$ 不是向量组 $((2, 1, -3), (1, -2, 4))$ 的线性组合。要证明这一点，我们需要说明不存在标量 $a\_1, a\_2 \\in \\mathbf{F}$ 使得： $$ (17, -4, 5) = a\_1(2, 1, -3) + a\_2(1, -2, 4) $$ 这等价于求解下面的方程组： $$ \\begin{cases} 17 = 2a\_1 + a\_2 \\\\ -4 = a\_1 - 2a\_2 \\\\ 5 = -3a\_1 + 4a\_2 \\end{cases} $$ 从前两个方程可以解得 $a\_1=6, a\_2=5$。但将这两个值代入第三个方程，得到 $-3(6) + 4(5) = -18 + 20 = 2$，而 $2 \\neq 5$。因此，这个方程组无解，说明 $(17, -4, 5)$ 无法表示为给定向量组的线性组合。

**定义 2.5：张成空间 (span)** $V$ 中一组向量 $(v\_1, \\ldots, v\_m)$ 的所有线性组合构成的集合称为 $(v\_1, \\ldots, v\_m)$ 的**张成空间**，记为 $\\text{span}(v\_1, \\ldots, v\_m)$。即： $$ \\text{span}(v\_1, \\ldots, v\_m) = {a\_1v\_1 + \\cdots + a\_m v\_m : a\_1, \\ldots, a\_m \\in \\mathbf{F}} $$ 我们规定，空向量组的张成空间为 ${0}$。

**命题 2.7：张成空间是包含这些向量的最小子空间** $V$ 中一组向量的张成空间是包含这组向量的最小子空间。

**证明：** 设 $(v\_1, \\ldots, v\_m)$ 是 $V$ 中的一组向量。 首先，我们需要证明 $\\text{span}(v\_1, \\ldots, v\_m)$ 是 $V$ 的一个子空间。根据子空间的三个条件：

1.  **加法单位元：** $0 = 0v\_1 + \\cdots + 0v\_m$，所以 $0 \\in \\text{span}(v\_1, \\ldots, v\_m)$。
2.  **加法封闭性：** 考虑两个在张成空间中的任意向量 $u = a\_1v\_1 + \\cdots + a\_m v\_m$ 和 $w = c\_1v\_1 + \\cdots + c\_m v\_m$。它们的和为： $$ u+w = (a\_1+c\_1)v\_1 + \\cdots + (a\_m+c\_m)v\_m $$ 这仍然是 $(v\_1, \\ldots, v\_m)$ 的一个线性组合，所以 $u+w \\in \\text{span}(v\_1, \\ldots, v\_m)$。
3.  **标量乘法封闭性：** 对于 $\\lambda \\in \\mathbf{F}$，有： $$ \\lambda u = \\lambda(a\_1v\_1 + \\cdots + a\_m v\_m) = (\\lambda a\_1)v\_1 + \\cdots + (\\lambda a\_m)v\_m $$ 这也属于张成空间。 因此，$\\text{span}(v\_1, \\ldots, v\_m)$ 是 $V$ 的一个子空间。

其次，对于每个 $j=1, \\ldots, m$，我们可以通过令 $a\_j=1$ 且其他标量为 $0$ 来得到 $v\_j$。所以，每个 $v\_j$ 都属于 $\\text{span}(v\_1, \\ldots, v\_m)$。 最后，任何包含 $(v\_1, \\ldots, v\_m)$ 的子空间，根据其加法封闭性和标量乘法封闭性，必然包含它们的所有线性组合。这意味着任何包含这组向量的子空间也必然包含 $\\text{span}(v\_1, \\ldots, v\_m)$。 综上所述，$\\text{span}(v\_1, \\ldots, v\_m)$ 是包含 $(v\_1, \\ldots, v\_m)$ 的最小子空间。证毕。

**定义 2.10：有限维向量空间 (finite-dimensional vector space)** 如果一个向量空间可以由其内的一组有限多个向量张成，则称这个向量空间是**有限维的**。

**定义 2.15：无限维向量空间 (infinite-dimensional vector space)** 不能被有限多个向量张成的向量空间称为**无限维的**。

**例子 2.16：P(F) 是无限维的** $\\mathbf{P}(\\mathbf{F})$ 表示由 $\\mathbf{F}$ 上所有多项式构成的向量空间。 **证明：** 假设 $\\mathbf{P}(\\mathbf{F})$ 是有限维的，那么它可由一组有限的多项式 $(p\_1, \\ldots, p\_m)$ 张成。设 $d$ 是这些多项式中次数的最大值。那么，任何这些多项式的线性组合，其次数都不可能超过 $d$。 然而，多项式 $z^{d+1}$ 属于 $\\mathbf{P}(\\mathbf{F})$，但它的次数是 $d+1$，所以它不可能写成 $(p\_1, \\ldots, p\_m)$ 的线性组合。 这与我们的假设相矛盾。因此，$\\mathbf{P}(\\mathbf{F})$ 是无限维的。证毕。

#### **线性无关**

张成的概念告诉我们如何从一组向量生成一个空间。现在我们来关注生成这个空间的“效率”。我们希望用尽可能少的向量来张成一个空间，这就引出了线性无关的概念。

**定义 2.17：线性无关 (linearly independent)** $V$ 中一组向量 $(v\_1, \\ldots, v\_m)$ 称为**线性无关的**，如果使得 $$ a\_1v\_1 + \\cdots + a\_m v\_m = 0 $$ 成立的唯一一组标量是 $a\_1 = \\cdots = a\_m = 0$。

**定义 2.19：线性相关 (linearly dependent)** $V$ 中一组向量 $(v\_1, \\ldots, v\_m)$ 如果不是线性无关的，就称为**线性相关的**。也就是说，存在不全为零的标量 $a\_1, \\ldots, a\_m \\in \\mathbf{F}$ 使得 $a\_1v\_1 + \\cdots + a\_m v\_m = 0$。

**例子 2.20：线性相关**

*   在 $\\mathbf{F}^3$ 中，向量组 $((2, 3, 1), (1, -1, 2), (7, 3, 8))$ 是线性相关的，因为 $$ 2(2, 3, 1) + 3(1, -1, 2) - 1(7, 3, 8) = (4+3-7, 6-3-3, 2+6-8) = (0, 0, 0) $$ 这里我们找到了不全为零的系数 $(2, 3, -1)$。
*   任何包含零向量的向量组都是线性相关的。

线性相关性有一个非常直观的解释：一个向量组是线性相关的，当且仅当其中至少有一个向量可以被其他向量表示出来。下面这个引理精确地描述了这一点。

**命题 2.21：线性相关引理** 如果 $(v\_1, \\ldots, v\_m)$ 是 $V$ 中的一个线性相关组，并且 $v\_1 \\neq 0$，那么存在 $j \\in {2, \\ldots, m}$ 使得： (a) $v\_j \\in \\text{span}(v\_1, \\ldots, v\_{j-1})$ (b) 如果从 $(v\_1, \\ldots, v\_m)$ 中去掉第 $j$ 项，则剩下向量组的张成空间与原向量组的张成空间相同。

**证明：** (a) 因为向量组是线性相关的，所以存在不全为零的标量 $a\_1, \\ldots, a\_m$ 使得 $a\_1v\_1 + \\cdots + a\_m v\_m = 0$。 让 $j$ 是使得 $a\_j \\neq 0$ 的最大下标。因为 $v\_1 \\neq 0$，所以 $j$ 必须大于1。于是我们可以把 $v\_j$ 解出来： $$ v\_j = -\\frac{a\_1}{a\_j}v\_1 - \\cdots - \\frac{a\_{j-1}}{a\_j}v\_{j-1} $$ 这表明 $v\_j$ 是前面向量 $(v\_1, \\ldots, v\_{j-1})$ 的线性组合，即 $v\_j \\in \\text{span}(v\_1, \\ldots, v\_{j-1})$。

(b) 设 $U = \\text{span}(v\_1, \\ldots, v\_{j-1}, v\_{j+1}, \\ldots, v\_m)$。显然 $U \\subseteq \\text{span}(v\_1, \\ldots, v\_m)$。 为了证明反向包含，我们任取一个原张成空间中的向量 $v = c\_1v\_1 + \\cdots + c\_m v\_m$。我们可以用(a)中的表达式替换 $v\_j$： $$ v = \\sum\_{k=1}^{j-1} c\_k v\_k + c\_j(-\\frac{a\_1}{a\_j}v\_1 - \\cdots - \\frac{a\_{j-1}}{a\_j}v\_{j-1}) + \\sum\_{k=j+1}^{m} c\_k v\_k $$ 整理后， $v$ 可以表示为除 $v\_j$ 之外的其他向量的线性组合。因此 $v \\in U$。 所以两个张成空间相等。证毕。

现在我们来证明本节最重要的一个结果，它联系了线性无关组和张成组的长度。

**定理 2.23：线性无关组的长度 $\\le$ 张成组的长度** 在有限维向量空间中，任意线性无关向量组的长度都小于或等于任意张成向量组的长度。

**证明：** 设 $V$ 是有限维的，$(u\_1, \\ldots, u\_m)$ 是 $V$ 中一个线性无关组，$(w\_1, \\ldots, w\_n)$ 是 $V$ 的一个张成组。我们要证明 $m \\le n$。

**第1步：** 因为 $(w\_1, \\ldots, w\_n)$ 张成 $V$，所以向量 $u\_1$ 可以写成它们的线性组合。因此，向量组 $(u\_1, w\_1, \\ldots, w\_n)$ 是线性相关的。 根据线性相关引理，我们可以从这个组中去掉一个 $w\_j$，使得剩下的向量组仍然张成 $V$。为方便起见（可以重新排序），我们假设去掉的是 $w\_1$。现在我们有了一个新的张成组 $(u\_1, w\_2, \\ldots, w\_n)$，长度仍为 $n$。

**第j步：** 假设我们已经进行了 $j-1$ 步，得到了一个张成组 $B\_{j-1}$，它的前 $j-1$ 个向量是 $(u\_1, \\ldots, u\_{j-1})$，后面是 $n-(j-1)$ 个来自 $w$ 组的向量。 现在我们将 $u\_j$ 加入这个张成组 $B\_{j-1}$。因为 $B\_{j-1}$ 张成 $V$，所以 $u\_j$ 是 $B\_{j-1}$ 中向量的线性组合。因此，新组 $(u\_1, \\ldots, u\_j, \\ldots)$ 是线性相关的。 根据线性相关引理，其中有一个向量是它前面向量的线性组合。这个向量不可能是 $u\_k$（其中 $k \\le j$），因为 $(u\_1, \\ldots, u\_m)$ 是线性无关的。所以，这个向量必定是某个 $w$ 向量。我们可以把这个 $w$ 向量去掉，剩下的向量组仍然张成 $V$。 我们就得到了一个新的长度为 $n$ 的张成组 $B\_j$，它以 $(u\_1, \\ldots, u\_j)$ 开头。

这个过程可以一直进行下去。每一步，我们添加一个 $u$ 向量，并去掉一个 $w$ 向量。这个过程能进行多少步呢？最多只能进行 $m$ 步（因为我们只有 $m$ 个 $u$ 向量）。每一步都会用掉一个 $w$ 向量。如果我们有 $m > n$，那么在第 $n$ 步之后，所有的 $w$ 向量都已经被替换掉了，我们会得到一个张成组 $(u\_1, \\ldots, u\_n)$。但这意味着 $u\_{n+1}$ (如果 $m>n$) 是 $(u\_1, \\ldots, u\_n)$ 的线性组合，这与 $(u\_1, \\ldots, u\_m)$ 线性无关的假设相矛盾。 因此，我们的假设 $m>n$ 必定是错误的。所以必须有 $m \\le n$。证毕。

### **2.B 基**

我们现在将张成和线性无关这两个概念结合起来，得到线性代数中最核心的概念之一：**基**。

**定义 2.27：基 (basis)** $V$ 中的一个向量组 $(v\_1, \\ldots, v\_n)$ 如果既是线性无关的，又张成 $V$，则称它为 $V$ 的一个**基**。

**例子 2.28：基的例子** (a) $((1,0,\\ldots,0), (0,1,0,\\ldots,0), \\ldots, (0,\\ldots,0,1))$ 是 $\\mathbf{F}^n$ 的一个基，称为**标准基**。 (b) $((1,2), (3,5))$ 是 $\\mathbf{F}^2$ 的一个基。它是线性无关的，并且张成 $\\mathbf{F}^2$（验证留作练习）。 (g) $(1, z, z^2, \\ldots, z^m)$ 是 $\\mathbf{P}\_m(\\mathbf{F})$ 的一个基。

基的一个美妙特性是，它为向量空间中的每一个向量提供了唯一的“地址”或“坐标”。

**命题 2.29：基的判定准则** $V$ 中的一个向量组 $(v\_1, \\ldots, v\_n)$ 是 $V$ 的一个基，当且仅当 $V$ 中的每个向量 $v$ 都可以被**唯一地**写成如下线性组合的形式： $$ v = a\_1v\_1 + \\cdots + a\_n v\_n $$

**证明：** $(\\Rightarrow)$ 假设 $(v\_1, \\ldots, v\_n)$ 是一个基。任取 $v \\in V$。因为基张成 $V$，所以 $v$ 至少有一种表示方法。假设有两种表示方法： $$ v = a\_1v\_1 + \\cdots + a\_n v\_n $$ $$ v = c\_1v\_1 + \\cdots + c\_n v\_n $$ 两式相减得到： $$ 0 = (a\_1-c\_1)v\_1 + \\cdots + (a\_n-c\_n)v\_n $$ 因为 $(v\_1, \\ldots, v\_n)$ 是线性无关的，所以所有系数必须为零，即 $a\_j - c\_j = 0$ for all $j$。因此 $a\_j = c\_j$，表示方法是唯一的。

$(\\Leftarrow)$ 假设每个 $v \\in V$ 都有唯一的表示。

*   **张成性：** 定义已经说明了每个向量都能被表示，所以 $(v\_1, \\ldots, v\_n)$ 张成 $V$。
*   **线性无关性：** 考虑 $0$ 的表示。一种显然的表示是 $0 = 0v\_1 + \\cdots + 0v\_n$。根据唯一性假设，这是唯一的表示方法。因此，任何形如 $a\_1v\_1 + \\cdots + a\_n v\_n = 0$ 的表达式都必须有 $a\_1 = \\cdots = a\_n = 0$。所以 $(v\_1, \\ldots, v\_n)$ 是线性无关的。 证毕。

现在我们知道什么是基了，但基一定存在吗？下面的命题回答了这个问题。

**命题 2.31：张成组可以缩减为基** 如果一个向量组张成 $V$，那么它可以被缩减为 $V$ 的一个基。

**证明：** 设 $(v\_1, \\ldots, v\_m)$ 张成 $V$。我们按以下步骤构建一个基： **第1步：** 如果 $v\_1=0$，就去掉它。如果 $v\_1 \\neq 0$，就保留它。 **第j步：** 假设我们已经处理了前 $j-1$ 个向量。考察 $v\_j$。如果 $v\_j \\in \\text{span}(v\_1, \\ldots, v\_{j-1})$（这里指的是保留下来的向量），我们就去掉 $v\_j$。否则，我们保留 $v\_j$。 我们对所有向量重复这个过程。最终剩下的向量组记为 $B$。

*   **张成性：** 每当我们去掉一个向量 $v\_j$ 时，是因为它已经在前面向量的张成空间里。根据线性相关引理，去掉它不会改变张成空间。所以，最终的组 $B$ 仍然张成 $V$。
*   **线性无关性：** 根据我们的构建方法，任何被保留的向量都不在它前面向量的张成空间里。再次利用线性相关引理的逆否命题，这保证了组 $B$ 是线性无关的。 因此，$B$ 是 $V$ 的一个基。证毕。

**推论 2.32：每个有限维向量空间都有基** 这直接由命题 2.31 得出，因为有限维向量空间根据定义就有一个有限的张成组。

**命题 2.33：线性无关组可以扩充为基** 在有限维向量空间中，任意线性无关的向量组都可以扩充为 $V$ 的一个基。

**证明：** 设 $(u\_1, \\ldots, u\_m)$ 是 $V$ 中的一个线性无关组。因为 $V$ 是有限维的，所以存在一个张成组 $(w\_1, \\ldots, w\_n)$。 考虑向量组 $(u\_1, \\ldots, u\_m, w\_1, \\ldots, w\_n)$。这个组显然张成 $V$（因为它包含了原来的张成组）。 现在，我们应用命题 2.31 的缩减过程来从这个新组中得到一个基。在这个过程中，任何 $u\_j$ 都不会被去掉，因为 $(u\_1, \\ldots, u\_m)$ 是线性无关的，所以没有一个 $u\_j$ 会在它前面的 $u$ 向量的张成空间里。 因此，我们得到的基会包含所有的 $u\_j$，从而完成了对原线性无关组的扩充。证毕。

### **2.C 维数**

我们已经知道每个有限维向量空间都有基。那么，一个向量空间的不同基，它们的长度是否相同呢？

**定理 2.35：任意两个基的长度相同** 在有限维向量空间中，任意两个基都有相同的长度。

**证明：** 设 $B\_1 = (v\_1, \\ldots, v\_m)$ 和 $B\_2 = (w\_1, \\ldots, w\_n)$ 是 $V$ 的两个基。

*   因为 $B\_1$ 是线性无关的，且 $B\_2$ 张成 $V$，根据定理 2.23，我们有 $m \\le n$。
*   因为 $B\_2$ 是线性无关的，且 $B\_1$ 张成 $V$，根据定理 2.23，我们有 $n \\le m$。 结合这两个不等式，我们必然得到 $m = n$。证毕。

这个定理非常重要，因为它保证了下面这个定义是合理的。

**定义 2.36：维数 (dimension)**

*   一个有限维向量空间 $V$ 的**维数**，记为 $\\dim V$，是它任意一个基的长度。
*   如果 $V$ 是无限维的，我们记 $\\dim V = \\infty$。

**例子 2.37：维数的例子**

*   $\\dim \\mathbf{F}^n = n$ (因为标准基有 $n$ 个向量)。
*   $\\dim \\mathbf{P}\_m(\\mathbf{F}) = m+1$ (因为基 $(1, z, \\ldots, z^m)$ 有 $m+1$ 个向量)。

维数这个概念非常强大，它可以帮助我们简化很多判断。

**命题 2.39：长度等于维数的线性无关组是基** 若 $V$ 是有限维的，且 $(v\_1, \\ldots, v\_n)$ 是 $V$ 中一个长度为 $\\dim V$ 的线性无关组，则它是一个基。

**证明：** 设 $(v\_1, \\ldots, v\_n)$ 是 $V$ 中的一个线性无关组，且 $n = \\dim V$。 根据命题 2.33，我们可以将这个组扩充为 $V$ 的一个基。但是，这个基的长度必须是 $\\dim V = n$。所以我们不能再添加任何向量。 因此，$(v\_1, \\ldots, v\_n)$ 本身就是一个基。证毕。

#### **子空间的维数**

**命题 2.43：子空间和的维数公式** 如果 $U\_1$ 和 $U\_2$ 是 $V$ 的两个有限维子空间，那么 $$ \\dim(U\_1 + U\_2) = \\dim U\_1 + \\dim U\_2 - \\dim(U\_1 \\cap U\_2) $$

**证明：** 设 $\\dim(U\_1 \\cap U\_2) = m$，并设 $(u\_1, \\ldots, u\_m)$ 是 $U\_1 \\cap U\_2$ 的一个基。 因为 $(u\_1, \\ldots, u\_m)$ 是 $U\_1$ 中的一个线性无关组，我们可以将其扩充为 $U\_1$ 的一个基：$(u\_1, \\ldots, u\_m, v\_1, \\ldots, v\_j)$。所以 $\\dim U\_1 = m+j$。 同样，我们可以将其扩充为 $U\_2$ 的一个基：$(u\_1, \\ldots, u\_m, w\_1, \\ldots, w\_k)$。所以 $\\dim U\_2 = m+k$。 现在我们来证明向量组 $B = (u\_1, \\ldots, u\_m, v\_1, \\ldots, v\_j, w\_1, \\ldots, w\_k)$ 是 $U\_1+U\_2$ 的一个基。

*   **张成性：** $U\_1+U\_2$ 中的任意向量可以写成 $u+w$ 的形式，其中 $u \\in U\_1, w \\in U\_2$。而 $u$ 和 $w$ 都可以用它们各自基的线性组合表示，所以 $u+w$ 可以用组 $B$ 中向量的线性组合表示。因此 $B$ 张成 $U\_1+U\_2$。
*   **线性无关性：** 假设 $$ \\sum a\_i u\_i + \\sum b\_i v\_i + \\sum c\_i w\_i = 0 $$ 移项得： $$ \\sum c\_i w\_i = -(\\sum a\_i u\_i + \\sum b\_i v\_i) $$ 等式左边在 $U\_2$ 中，右边在 $U\_1$ 中。所以这个向量在 $U\_1 \\cap U\_2$ 中。因此，它可以表示为 $(u\_1, \\ldots, u\_m)$ 的线性组合: $\\sum c\_i w\_i = \\sum d\_i u\_i$。 这导致 $\\sum c\_i w\_i - \\sum d\_i u\_i = 0$。因为 $(u\_1, \\ldots, u\_m, w\_1, \\ldots, w\_k)$ 是 $U\_2$ 的基（线性无关），所以所有的 $c\_i$ 和 $d\_i$ 都必须是0。 现在回到最初的等式，我们有 $\\sum a\_i u\_i + \\sum b\_i v\_i = 0$。因为 $(u\_1, \\ldots, u\_m, v\_1, \\ldots, v\_j)$ 是 $U\_1$ 的基（线性无关），所以所有的 $a\_i$ 和 $b\_i$ 也必须是0。 因此，组 $B$ 是线性无关的。

既然 $B$ 是 $U\_1+U\_2$ 的基，那么 $$ \\begin{aligned} \\dim(U\_1 + U\_2) &= m + j + k \\\\ &= (m+j) + (m+k) - m \\\\ &= \\dim U\_1 + \\dim U\_2 - \\dim(U\_1 \\cap U\_2) \\end{aligned} $$ 证毕。

**例子：维数公式的应用**

在 $\\mathbf{R}^3$ 中，设 $U\_1$ 是 xy-平面， $U\_2$ 是 yz-平面。 $U\_1 = {(x, y, 0) : x, y \\in \\mathbf{R}}$，一个基是 $((1,0,0), (0,1,0))$，所以 $\\dim U\_1 = 2$。 $U\_2 = {(0, y, z) : y, z \\in \\mathbf{R}}$，一个基是 $((0,1,0), (0,0,1))$，所以 $\\dim U\_2 = 2$。 它们的交集 $U\_1 \\cap U\_2$ 是 y-轴: ${(0, y, 0) : y \\in \\mathbf{R}}$。一个基是 $((0,1,0))$，所以 $\\dim(U\_1 \\cap U\_2) = 1$。 它们的和 $U\_1+U\_2$ 包含所有形如 $(x, y\_1, 0) + (0, y\_2, z) = (x, y\_1+y\_2, z)$ 的向量，这显然是整个 $\\mathbf{R}^3$。所以 $\\dim(U\_1+U\_2) = 3$。 我们来验证公式： $$ \\dim U\_1 + \\dim U\_2 - \\dim(U\_1 \\cap U\_2) = 2 + 2 - 1 = 3 $$ 这与 $\\dim(U\_1+U\_2)$ 相符。

**第二讲小结**

今天我们深入探讨了有限维向量空间的结构。

1.  我们定义了**线性组合**和**张成空间**，这是从少数向量生成整个空间的方法。
2.  我们引入了**线性无关**的概念，它刻画了生成空间的向量组的“效率”。
3.  结合这两个概念，我们得到了**基**的定义——一个张成空间的线性无关组。基为空间中的每个向量提供了唯一的坐标。
4.  我们证明了任何有限维向量空间都有基，并且任意两个基的长度都相同，这使得**维数**的定义成为可能。
5.  最后，我们推导了关于子空间维数的重要公式，特别是子空间和的维数公式。
