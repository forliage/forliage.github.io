# 线性代数01:向量空间

线性代数的核心研究对象是“有限维向量空间上的线性映射”。为了透彻地理解这个核心概念，我们必须首先对“向量空间”本身有一个清晰、准确的认识。本章的目的就是建立起向量空间的基本概念和理论。

### **1\. $\\mathbf{R}^n$ 与 $\\mathbf{C}^n$**

在正式给出向量空间的抽象定义之前，我们先从一些熟悉的例子入手，这些例子将为我们后续的抽象化提供坚实的直观基础。我们首先需要回顾一下实数和复数。

#### **复数**

我们已经很熟悉实数集 $\\mathbf{R}$ 的基本性质。复数的发明使得我们可以对负数取平方根。

**定义 1.1：复数 (complex number)**

*   一个**复数**是一个有序对 $(a, b)$，其中 $a, b \\in \\mathbf{R}$。但我们通常将其写作 $a + bi$。
*   所有复数构成的集合记为 $\\mathbf{C}$，即： $$ \\mathbf{C} = {a + bi : a, b \\in \\mathbf{R}} $$
*   $\\mathbf{C}$ 上的**加法**和**乘法**定义如下： 对于 $a, b, c, d \\in \\mathbf{R}$：
    *   加法：$(a + bi) + (c + di) = (a + c) + (b + d)i$
    *   乘法：$(a + bi)(c + di) = (ac - bd) + (ad + bc)i$

**例子 1.2：复数乘法**

计算 $(2+3i)(4+5i)$。

**解：** 根据乘法定义： $$ \\begin{aligned} (2+3i)(4+5i) &= 2 \\cdot 4 - 3 \\cdot 5 + (2 \\cdot 5 + 3 \\cdot 4)i \\\\ &= 8 - 15 + (10 + 12)i \\\\ &= -7 + 22i \\end{aligned} $$

**定义 1.3：复数的算术性质**

复数具有以下重要的算术性质，对于任意 $\\alpha, \\beta, \\lambda \\in \\mathbf{C}$：

*   **交换性 (commutativity):** $\\alpha + \\beta = \\beta + \\alpha$, $\\alpha\\beta = \\beta\\alpha$
*   **结合性 (associativity):** $(\\alpha + \\beta) + \\lambda = \\alpha + (\\beta + \\lambda)$, $(\\alpha\\beta)\\lambda = \\alpha(\\beta\\lambda)$
*   **单位元 (identities):** 存在 $0, 1 \\in \\mathbf{C}$ 使得 $\\lambda + 0 = \\lambda$, $\\lambda \\cdot 1 = \\lambda$
*   **加法逆元 (additive inverse):** 对每个 $\\alpha \\in \\mathbf{C}$，都存在唯一的 $-\\alpha \\in \\mathbf{C}$ 使得 $\\alpha + (-\\alpha) = 0$
*   **乘法逆元 (multiplicative inverse):** 对每个 $\\alpha \\in \\mathbf{C}, \\alpha \\neq 0$，都存在唯一的 $\\alpha^{-1} \\in \\mathbf{C}$ 使得 $\\alpha\\alpha^{-1} = 1$
*   **分配性质 (distributive property):** $\\lambda(\\alpha + \\beta) = \\lambda\\alpha + \\lambda\\beta$

**记号 1.6：$\\mathbf{F}$** 为了方便叙述，我们用符号 $\\mathbf{F}$ 来表示 $\\mathbf{R}$ 或 $\\mathbf{C}$。$\\mathbf{F}$ 中的元素称为**标量 (scalar)**。

#### **$\\mathbf{F}^n$**

我们最熟悉的向量空间例子是 $\\mathbf{R}^2$（平面）和 $\\mathbf{R}^3$（三维空间）。现在我们将这个概念推广到由 $\\mathbf{F}$ 中元素构成的 n 元组。

**定义 1.8：组 (list) 与 n元组 (n-tuple)** 长度为 $n$ 的**组**是 $n$ 个有顺序的元素。长度为 $n$ 的组也称为 **n元组**。其形式如下： $$ (x\_1, x\_2, \\ldots, x\_n) $$

**定义 1.10：$\\mathbf{F}^n$** $\\mathbf{F}^n$ 是由 $\\mathbf{F}$ 中元素构成的所有长度为 $n$ 的组的集合： $$ \\mathbf{F}^n = {(x\_1, \\ldots, x\_n) : x\_j \\in \\mathbf{F} \\text{ for } j=1, \\ldots, n} $$ $\\mathbf{F}^n$ 中的一个元素 $(x\_1, \\ldots, x\_n)$ 的第 $j$ 个坐标是 $x\_j$。

**定义 1.12：$\\mathbf{F}^n$ 中的加法 (addition in $\\mathbf{F}^n$)** $\\mathbf{F}^n$ 中的加法是按坐标位相加的： $$ (x\_1, \\ldots, x\_n) + (y\_1, \\ldots, y\_n) = (x\_1+y\_1, \\ldots, x\_n+y\_n) $$

**定义 1.17：$\\mathbf{F}^n$ 中的标量乘法 (scalar multiplication in $\\mathbf{F}^n$)** $\\mathbf{F}^n$ 中的标量乘法定义为标量与每个坐标相乘： 对于 $\\lambda \\in \\mathbf{F}$ 和 $(x\_1, \\ldots, x\_n) \\in \\mathbf{F}^n$： $$ \\lambda(x\_1, \\ldots, x\_n) = (\\lambda x\_1, \\ldots, \\lambda x\_n) $$

**例子：$\\mathbf{R}^3$ 中的运算**

在 $\\mathbf{R}^3$ 中，设向量 $v = (1, -2, 4)$ 和 $w = (5, 0, -3)$，标量 $\\lambda = 2$。

**解：**

*   **向量加法：** $$ v + w = (1, -2, 4) + (5, 0, -3) = (1+5, -2+0, 4-3) = (6, -2, 1) $$
*   **标量乘法：** $$ \\lambda v = 2(1, -2, 4) = (2 \\cdot 1, 2 \\cdot (-2), 2 \\cdot 4) = (2, -4, 8) $$

$\\mathbf{F}^n$ 以及在其上定义的加法和标量乘法，是我们学习向量空间的出发点。接下来，我们将把这些观察到的性质抽象出来，形成向量空间的一般定义。

### **1.B 向量空间的定义**

现在，我们准备给出一个数学上更普适的定义。一个向量空间是一个集合，连同一个“加法”和一个“标量乘法”，并且这些运算满足一些合理的性质（公理）。

**定义 1.19：向量空间 (vector space)** 一个**向量空间**是集合 $V$ 以及其上的两种运算——**加法**（$V$ 中任意两个元素之和仍在 $V$ 中）和**标量乘法**（$\\mathbf{F}$ 中任意标量与 $V$ 中任意元素之积仍在 $V$ 中），并满足以下所有性质： 对所有 $u, v, w \\in V$ 和 $a, b \\in \\mathbf{F}$：

1.  **加法交换性 (commutativity):** $u+v = v+u$
2.  **加法结合性 (associativity):** $(u+v)+w = u+(v+w)$
3.  **加法单位元 (additive identity):** 存在一个元素 $0 \\in V$ 使得对所有 $v \\in V$ 都有 $v+0=v$
4.  **加法逆元 (additive inverse):** 对每个 $v \\in V$，都存在一个 $w \\in V$ 使得 $v+w=0$
5.  **乘法单位元 (multiplicative identity):** 对所有 $v \\in V$ 都有 $1v = v$
6.  **标量乘法结合性 (associativity of scalar multiplication):** $(ab)v = a(bv)$
7.  **分配性质 (distributive properties):**
    *   $a(u+v) = au+av$
    *   $(a+b)v = av+bv$

**定义 1.21：实向量空间与复向量空间**

*   定义在标量域 $\\mathbf{R}$ 上的向量空间称为**实向量空间 (real vector space)**。
*   定义在标量域 $\\mathbf{C}$ 上的向量空间称为**复向量空间 (complex vector space)**。

向量空间中的元素称为**向量 (vector)** 或**点 (point)**。

**向量空间的例子**

除了我们已经熟悉的 $\\mathbf{F}^n$，还有很多其他重要的向量空间。

**例子 1.23：函数空间 $\\mathbf{F}^S$** 设 $S$ 是一个非空集合。$\\mathbf{F}^S$ 表示从集合 $S$ 到 $\\mathbf{F}$ 的所有函数的集合。

*   **加法：** 对于 $f, g \\in \\mathbf{F}^S$，函数 $f+g$ 定义为： $$ (f+g)(x) = f(x) + g(x) \\quad \\text{对所有 } x \\in S $$
*   **标量乘法：** 对于 $\\lambda \\in \\mathbf{F}$ 和 $f \\in \\mathbf{F}^S$，函数 $\\lambda f$ 定义为： $$ (\\lambda f)(x) = \\lambda f(x) \\quad \\text{对所有 } x \\in S $$ 在这些运算下，$\\mathbf{F}^S$ 是一个向量空间。

**例子：验证一个具体函数空间**

考虑所有从区间 $\[0, 1\]$ 到 $\\mathbf{R}$ 的连续函数构成的集合，记作 $C(\[0, 1\])$。验证它是否是一个实向量空间。

**解：** 我们需要验证 $C(\[0, 1\])$ 在上述函数加法和标量乘法下满足向量空间的所有公理。

*   **加法单位元：** 零函数 $f\_0(x) = 0$ 对所有 $x \\in \[0, 1\]$ 都是连续的。对于任意 $g \\in C(\[0, 1\])$，有 $(g+f\_0)(x) = g(x) + 0 = g(x)$，所以 $f\_0$ 是加法单位元。
*   **加法封闭性：** 如果 $f$ 和 $g$ 都是连续函数，那么它们的和 $f+g$ 也是一个连续函数。所以 $f+g \\in C(\[0, 1\])$。
*   **标量乘法封闭性：** 如果 $f$ 是连续函数，$\\lambda \\in \\mathbf{R}$，那么 $\\lambda f$ 也是一个连续函数。所以 $\\lambda f \\in C(\[0, 1\])$。
*   其他的性质（交换性、结合性、分配性等）都继承自实数 $\\mathbf{R}$ 的相应性质。例如，加法交换性成立是因为对任意 $x \\in \[0, 1\]$，$(f+g)(x) = f(x)+g(x) = g(x)+f(x) = (g+f)(x)$。

因此，$C(\[0, 1\])$ 是一个实向量空间。

**向量空间的基本性质**

从向量空间的公理可以推导出一些基本但非常重要的性质。

**命题 1.25：加法单位元唯一** 在一个向量空间中，加法单位元是唯一的。

**证明：** 假设 $0$ 和 $0'$ 都是 $V$ 中的加法单位元。那么： $$ 0' = 0' + 0 \\quad (\\text{因为 } 0 \\text{ 是加法单位元}) $$ $$ 0' + 0 = 0 + 0' \\quad (\\text{根据加法交换性}) $$ $$ 0 + 0' = 0 \\quad (\\text{因为 } 0' \\text{ 是加法单位元}) $$ 因此，我们有 $0' = 0$。证毕。

**命题 1.26：加法逆元唯一** 向量空间中的每个元素都有唯一的加法逆元。

**证明：** 设 $v \\in V$，并假设 $w$ 和 $w'$ 都是 $v$ 的加法逆元。则 $v+w=0$ 且 $v+w'=0$。我们有： $$ \\begin{aligned} w &= w + 0 \\\\ &= w + (v + w') \\\\ &= (w + v) + w' \\\\ &= (v + w) + w' \\\\ &= 0 + w' \\\\ &= w' \\end{aligned} $$ 因此 $w = w'$。证毕。我们将 $v$ 的加法逆元记为 $-v$。

**命题 1.29：数0乘向量** 对任意 $v \\in V$，有 $0v = 0$。

**证明：** $$ \\begin{aligned} 0v &= (0+0)v \\\\ &= 0v + 0v \\quad (\\text{分配性质}) \\end{aligned} $$ 在等式两边同时加上 $0v$ 的加法逆元 $-(0v)$，我们得到： $$ 0v + (-(0v)) = (0v + 0v) + (-(0v)) $$ $$ 0 = 0v + (0v + (-(0v))) $$ $$ 0 = 0v + 0 $$ $$ 0 = 0v $$ 证毕。

### **1.C 子空间**

在研究一个向量空间时，我们常常会对它的一些满足特定条件的子集感兴趣。如果这些子集本身也构成向量空间，我们就称之为**子空间**。

**定义 1.32：子空间 (subspace)** 设 $V$ 是一个向量空间。$V$ 的一个子集 $U$ 如果在使用与 $V$ 相同的加法和标量乘法下也构成一个向量空间，则称 $U$ 是 $V$ 的一个**子空间**。

要判断一个子集 $U$ 是否为子空间，我们不必去验证所有的向量空间公理。因为很多公理（如交换性、结合性）都是自动继承的。我们只需要验证以下三个关键条件。

**命题 1.34：子空间的条件** 一个向量空间 $V$ 的子集 $U$ 是 $V$ 的子空间，当且仅当 $U$ 满足以下三个条件：

1.  **加法单位元 (additive identity):** $0 \\in U$
2.  **加法封闭性 (closed under addition):** 如果 $u, w \\in U$，则 $u+w \\in U$
3.  **标量乘法封闭性 (closed under scalar multiplication):** 如果 $a \\in \\mathbf{F}$ 且 $u \\in U$，则 $au \\in U$

**例子 1.35：子空间的判定**

判断以下集合是否为 $\\mathbf{F}^3$ 的子空间： (a) $U\_1 = {(x\_1, x\_2, x\_3) \\in \\mathbf{F}^3 : x\_1 + 2x\_2 + 3x\_3 = 0}$ (b) $U\_2 = {(x\_1, x\_2, x\_3) \\in \\mathbf{F}^3 : x\_1 + 2x\_2 + 3x\_3 = 4}$

**解：** (a) 对于 $U\_1$:

1.  **加法单位元:** 取 $(x\_1, x\_2, x\_3) = (0, 0, 0)$，则 $0 + 2(0) + 3(0) = 0$。所以 $0 = (0, 0, 0) \\in U\_1$。
2.  **加法封闭性:** 设 $u=(x\_1, x\_2, x\_3)$ 和 $v=(y\_1, y\_2, y\_3)$ 都是 $U\_1$ 中的元素。则 $x\_1 + 2x\_2 + 3x\_3 = 0$ 且 $y\_1 + 2y\_2 + 3y\_3 = 0$。它们的和是 $u+v = (x\_1+y\_1, x\_2+y\_2, x\_3+y\_3)$。我们检验其分量是否满足条件： $$ (x\_1+y\_1) + 2(x\_2+y\_2) + 3(x\_3+y\_3) = (x\_1+2x\_2+3x\_3) + (y\_1+2y\_2+3y\_3) = 0 + 0 = 0 $$ 所以 $u+v \\in U\_1$。
3.  **标量乘法封闭性:** 设 $a \\in \\mathbf{F}$ 且 $u=(x\_1, x\_2, x\_3) \\in U\_1$。则 $au = (ax\_1, ax\_2, ax\_3)$。我们检验其分量： $$ (ax\_1) + 2(ax\_2) + 3(ax\_3) = a(x\_1+2x\_2+3x\_3) = a(0) = 0 $$ 所以 $au \\in U\_1$。 因此，$U\_1$ 是 $\\mathbf{F}^3$ 的一个子空间。

(b) 对于 $U\_2$:

1.  **加法单位元:** 取 $(x\_1, x\_2, x\_3) = (0, 0, 0)$，则 $0 + 2(0) + 3(0) = 0 \\neq 4$。所以 $0 \\notin U\_2$。 由于条件1不满足，$U\_2$ 不是 $\\mathbf{F}^3$ 的子空间。

#### **子空间的和与直和**

我们可以通过已有的子空间来构造新的子空间。

**定义 1.36：子集的和 (sum of subsets)** 设 $U\_1, \\ldots, U\_m$ 是 $V$ 的子集。这些子集的**和**，记作 $U\_1 + \\cdots + U\_m$，定义为所有可能元素之和构成的集合： $$ U\_1 + \\cdots + U\_m = {u\_1 + \\cdots + u\_m : u\_1 \\in U\_1, \\ldots, u\_m \\in U\_m} $$ 如果 $U\_1, \\ldots, U\_m$ 都是 $V$ 的子空间，那么它们的和也是 $V$ 的一个子空间。

有时候，我们不仅关心一个向量能否表示成子空间中向量的和，更关心这种表示是否**唯一**。这就引出了**直和**的概念。

**定义 1.40：直和 (direct sum)** 设 $U\_1, \\ldots, U\_m$ 是 $V$ 的子空间。

*   和 $U\_1 + \\cdots + U\_m$ 称为**直和**，如果 $U\_1 + \\cdots + U\_m$ 中的每个元素都能**唯一地**写成 $u\_1 + \\cdots + u\_m$（其中每个 $u\_j \\in U\_j$）的形式。
*   我们使用符号 $U\_1 \\oplus \\cdots \\oplus U\_m$ 来表示直和。

如何判断一个和是不是直和呢？有一个等价的、更易于检验的条件。

**命题 1.44：直和的条件** 设 $U\_1, \\ldots, U\_m$ 是 $V$ 的子空间。和 $U\_1 + \\cdots + U\_m$ 是直和的当且仅当，将 $0$ 写成和 $u\_1 + \\cdots + u\_m$（其中 $u\_j \\in U\_j$）的唯一方式是令所有 $u\_j$ 都等于 $0$。

对于两个子空间的情形，这个条件可以被进一步简化。

**命题 1.45：两个子空间的直和** 设 $U$ 和 $W$ 是 $V$ 的子空间。则 $U+W$ 是直和当且仅当 $U \\cap W = {0}$。

**例子：直和的判定**

在 $\\mathbf{R}^3$ 中，设 $U = {(x, y, 0) : x, y \\in \\mathbf{R}}$ (xy-平面) $W = {(0, 0, z) : z \\in \\mathbf{R}}$ (z-轴) $X = {(0, y, y) : y \\in \\mathbf{R}}$ (在 yz-平面上的一条直线)

判断 $U+W$ 和 $U+X$ 是否为直和。

**解：**

*   **对于 $U+W$:** 我们需要计算它们的交集 $U \\cap W$。 如果一个向量 $v=(a, b, c)$ 同时在 $U$ 和 $W$ 中，那么：
    
    *   因为它在 $U$ 中，所以 $c=0$。
    *   因为它在 $W$ 中，所以 $a=0, b=0$。 因此，唯一满足条件的向量是 $(0, 0, 0)$，即 $U \\cap W = {0}$。 根据命题 1.45，$U+W$ 是直和，我们可以写作 $U \\oplus W$。（实际上，$\\mathbf{R}^3 = U \\oplus W$）
*   **对于 $U+X$:** 我们需要计算它们的交集 $U \\cap X$。 如果一个向量 $v=(a, b, c)$ 同时在 $U$ 和 $X$ 中，那么：
    
    *   因为它在 $U$ 中，所以 $c=0$。
    *   因为它在 $X$ 中，所以 $a=0$ 且 $c=b$。 结合这两个条件，我们得到 $a=0, c=0, b=c=0$。所以交集仍然是 $U \\cap X = {0}$。 因此，$U+X$ 也是直和，我们可以写作 $U \\oplus X$。

**第一讲小结**

今天我们学习了线性代数中最基本的概念——向量空间。

1.  我们从具体的例子 $\\mathbf{R}^n$ 和 $\\mathbf{C}^n$ 出发，了解了向量加法和标量乘法的基本运算法则。
2.  然后，我们将这些性质抽象出来，给出了向量空间普适的公理化定义，并看到了一些更广泛的例子，如函数空间。
3.  最后，我们引入了子空间的概念，它是研究向量空间结构的重要工具，并学习了如何通过子空间的和与直和来构造和分解向量空间。
