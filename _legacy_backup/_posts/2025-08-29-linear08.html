线性代数08:复向量空间上的算子

在本章中，我们将始终采用以下假定：

  • $V$ 是一个有限维非零的复向量空间（即 $\mathbf{F}=\mathbf{C}$）。

8.A 广义本征向量和幂零算子

对于一个不可对角化的算子 $T$，意味着它的本征向量不足以张成整个空间 $V$。为了分解 $V$，我们需要寻找一种更“广义”的本征向量。

递增的零空间序列

命题 8.2 & 8.3：零空间序列的性质 设 $T \in \mathcal{L}(V)$。则 $$ {0} = \text{null } T^0 \subseteq \text{null } T^1 \subseteq \text{null } T^2 \subseteq \cdots $$ 并且，一旦这个序列中有相邻两项相等，即 $\text{null } T^m = \text{null } T^{m+1}$，那么此后的所有项也都相等： $$\text{null } T^m = \text{null } T^{m+1} = \text{null } T^{m+2} = \cdots$$

证明： 若 $v \in \text{null } T^k$，则 $T^k v = 0$。于是 $T^{k+1}v = T(T^k v) = T(0) = 0$，所以 $v \in \text{null } T^{k+1}$。这证明了序列是递增的。 现在假设 $\text{null } T^m = \text{null } T^{m+1}$。我们要证明 $\text{null } T^{m+1} = \text{null } T^{m+2}$。 我们已知 $\text{null } T^{m+1} \subseteq \text{null } T^{m+2}$。只需证明反向包含。 设 $v \in \text{null } T^{m+2}$。则 $T^{m+2}v = 0$，即 $T^{m+1}(Tv) = 0$。 这意味着 $Tv \in \text{null } T^{m+1}$。根据我们的假设，$\text{null } T^{m+1} = \text{null } T^m$，所以 $Tv \in \text{null } T^m$。 这意味着 $T^m(Tv)=0$，即 $T^{m+1}v=0$。 所以 $v \in \text{null } T^{m+1}$。证毕。

命题 8.4：零空间序列停止增长 设 $T \in \mathcal{L}(V)$ 且 $\dim V = n$。则 $$\text{null } T^n = \text{null } T^{n+1} = \text{null } T^{n+2} = \cdots$$

证明： 考虑序列 $\text{null } T^0 \subseteq \text{null } T^1 \subseteq \cdots$。 如果其中有严格包含关系，即 $\text{null } T^k \subsetneq \text{null } T^{k+1}$，那么 $\dim(\text{null } T^{k+1}) > \dim(\text{null } T^k)$，维数至少增加1。 这个序列的维数不可能无限增加，因为 $\dim(\text{null } T^k) \le n$。 所以，严格包含关系最多只能发生 $n$ 次。这意味着在序列 $\text{null } T^0, \ldots, \text{null } T^n$ 中，必然有相邻两项是相等的。 根据命题 8.3，一旦出现相等，后面的就都相等了。特别地，在第 $n$ 项之后，序列必然稳定。证毕。

命题 8.5：$V$ 分解为零空间与值域的直和 设 $T \in \mathcal{L}(V)$ 且 $n = \dim V$。则 $$ V = \text{null } T^n \oplus \text{range } T^n $$

证明： 设 $v \in \text{null } T^n \cap \text{range } T^n$。 因为 $v \in \text{range } T^n$，所以存在 $u \in V$ 使得 $v = T^n u$。 因为 $v \in \text{null } T^n$，所以 $T^n v = 0$。 结合起来，我们有 $T^n(T^n u) = T^{2n}u = 0$。所以 $u \in \text{null } T^{2n}$。 根据命题 8.4，$\text{null } T^{2n} = \text{null } T^n$。所以 $u \in \text{null } T^n$。 这意味着 $T^n u = 0$。但我们之前设 $v = T^n u$，所以 $v=0$。 因此，$\text{null } T^n \cap \text{range } T^n = {0}$。 根据线性映射基本定理，$\dim V = \dim(\text{null } T^n) + \dim(\text{range } T^n)$。 因为两个子空间的维数之和等于全空间的维数，且它们的交集为 ${0}$，所以它们的和是直和且等于 $V$。证毕。

广义本征向量

定义 8.9：广义本征向量 (generalized eigenvector) 设 $T \in \mathcal{L}(V)$ 且 $\lambda$ 是 $T$ 的一个本征值。一个向量 $v \in V$ 称为 $T$ 的对应于 $\lambda$ 的广义本征向量，如果存在正整数 $j$ 使得 $$ (T - \lambda I)^j v = 0 $$

定义 8.10：广义本征空间 (generalized eigenspace) $T$ 对应于本征值 $\lambda$ 的广义本征空间，记为 $G(\lambda, T)$，是 $T$ 对应于 $\lambda$ 的所有广义本征向量的集合，再加上零向量。 $$ G(\lambda, T) = \text{null}(T-\lambda I)^{\dim V} $$

幂零算子

定义 8.16：幂零算子 (nilpotent operator) 一个算子 $N \in \mathcal{L}(V)$ 称为幂零的，如果它的某个幂等于0。

命题 8.19：幂零算子的矩阵 设 $N \in \mathcal{L}(V)$ 是幂零的。则存在 $V$ 的一个基使得 $N$ 关于该基的矩阵是上三角的，且对角线上的元素全为0。

证明： 因为 $N$ 是幂零的，所以它的唯一本征值是0。 根据我们在第5讲证明的定理，任何复向量空间上的算子，都存在一个基使其矩阵是上三角的。对于 $N$ 来说，这个上三角矩阵的对角线元素必须是它的本征值，所以对角线元素全为0。证毕。

8.B 算子的分解

现在我们来到本章的核心定理，它表明任何复向量空间都可以分解为算子的广义本征空间的直和。

定理 8.21：复向量空间分解为广义本征空间 设 $T \in \mathcal{L}(V)$，$\lambda_1, \ldots, \lambda_m$ 是 $T$ 的所有互异本征值。则 (a) $V = G(\lambda_1, T) \oplus \cdots \oplus G(\lambda_m, T)$。 (b) 每个 $G(\lambda_j, T)$ 在 $T$ 下都是不变的。 (c) 在每个 $G(\lambda_j, T)$ 上，$T-\lambda_j I$ 都是幂零的。

证明： 我们对 $\dim V$ 进行归纳。 基础情形： $\dim V=1$ 时，$T$ 只有一个本征值 $\lambda_1$，此时 $G(\lambda_1, T) = V$，结论成立。 归纳假设： 假设对所有维数小于 $\dim V$ 的复向量空间，结论成立。 设 $\dim V = n$，$\lambda_1$ 是 $T$ 的一个本征值。令 $U = G(\lambda_1, T) = \text{null}(T-\lambda_1 I)^n$。 $U$ 在 $T$ 下是不变的。 令 $W = \text{range}(T-\lambda_1 I)^n$。$W$ 也在 $T$ 下是不变。 根据命题 8.5，我们有 $V = U \oplus W$。 考虑限制算子 $T|_W \in \mathcal{L}(W)$。它的本征值是什么？如果 $\mu$ 是 $T|_W$ 的本征值，那么它也必须是 $T$ 的本征值。但它不可能是 $\lambda_1$，否则对应的本征向量会同时在 $U$ 和 $W$ 中，而 $U \cap W = {0}$。 所以 $T|_W$ 的本征值是 $T$ 的本征值中除去 $\lambda_1$ 的那一部分，即 $\lambda_2, \ldots, \lambda_m$。 $\dim W < n$。根据归纳假设， $W$ 可以分解为 $T|_W$ 的广义本征空间的直和： $$ W = G(\lambda_2, T|_W) \oplus \cdots \oplus G(\lambda_m, T|_W) $$ 不难证明 $G(\lambda_j, T|_W) = G(\lambda_j, T)$ for $j=2, \ldots, m$。 因此，$V = U \oplus W = G(\lambda_1, T) \oplus G(\lambda_2, T) \oplus \cdots \oplus G(\lambda_m, T)$。 (a)得证。(b)和(c)由定义和构造过程直接得出。证毕。

这个定理的美妙之处在于，它将对任意算子 $T$ 的研究，分解为对每个限制算子 $T|_{G(\lambda_j, T)}$ 的研究。而每个这样的限制算子都可以写成 $\lambda_j I + N_j$ 的形式，其中 $N_j$ 是一个幂零算子。这样，我们就把对一般算子的研究转化为了对幂零算子的研究。

8.C 特征多项式和极小多项式

凯莱-哈密顿定理

定义 8.34：特征多项式 (characteristic polynomial) 设 $T \in \mathcal{L}(V)$，其互异本征值为 $\lambda_1, \ldots, \lambda_m$，对应的广义本征空间的维数（称为代数重数）为 $d_1, \ldots, d_m$。$T$ 的特征多项式定义为 $$ (z-\lambda_1)^{d_1} \cdots (z-\lambda_m)^{d_m} $$ 注意 $\sum d_j = \dim V$。

定理 8.37：凯莱-哈密顿定理 (Cayley-Hamilton Theorem) 设 $q$ 是 $T$ 的特征多项式。则 $q(T) = 0$。

证明： 令 $q(z) = (z-\lambda_1)^{d_1} \cdots (z-\lambda_m)^{d_m}$。 我们要证明 $q(T)v = 0$ 对所有 $v \in V$ 成立。 根据定理 8.21，我们只需对每个广义本征空间 $G(\lambda_j, T)$ 中的向量 $v$ 证明即可。 设 $v \in G(\lambda_j, T)$。我们有 $(T-\lambda_j I)^{d_j} v = 0$。 $q(T)$ 的因子中包含了 $(T-\lambda_j I)^{d_j}$。由于多项式作用于算子的乘法是可交换的，我们可以把这个因子移到最右边来作用于 $v$。 $$ q(T)v = \left( \prod_{k \neq j} (T-\lambda_k I)^{d_k} \right) (T-\lambda_j I)^{d_j} v = \left( \prod_{k \neq j} (T-\lambda_k I)^{d_k} \right) (0) = 0 $$ 证毕。

极小多项式

定义 8.40：极小多项式 (minimal polynomial) $T$ 的极小多项式是使得 $p(T)=0$ 的次数最小的首一多项式（最高次项系数为1）。

存在唯一性证明： 考虑算子组 $(I, T, T^2, \ldots, T^{n^2})$，其中 $n=\dim V$。$\mathcal{L}(V)$ 的维数是 $n^2$。这个组有 $n^2+1$ 个算子，所以是线性相关的。 因此，存在不全为零的系数使得 $a_0 I + \cdots + a_m T^m = 0$ ($m \le n^2$)。这表明存在一个多项式 $p$ 使得 $p(T)=0$。 在所有这样的多项式中，我们可以找到一个次数最小的。将它除以其首项系数，就得到了一个首一多项式。 唯一性：如果 $p_1, p_2$ 都是极小多项式，那么它们的次数相同。$p_1(T)-p_2(T)=0$。$(p_1-p_2)(T)=0$。但 $\text{deg}(p_1-p_2) < \text{deg } p_1$。这与 $p_1$ 的最小性矛盾，除非 $p_1-p_2=0$。

命题 8.49：极小多项式的零点 $T$ 的极小多项式的零点恰好是 $T$ 的本征值。

证明： 设 $p$ 是极小多项式，$\lambda$ 是本征值，对应本征向量 $v$。 $0 = p(T)v = p(\lambda)v$。因为 $v \neq 0$，所以 $p(\lambda)=0$。 反之，设 $\lambda$ 是 $p$ 的一个零点。则 $p(z)=(z-\lambda)q(z)$。 $0=p(T) = (T-\lambda I)q(T)$。 如果 $T-\lambda I$ 可逆，那么 $q(T)=0$，这与 $p$ 的最小性矛盾。 所以 $T-\lambda I$ 不可逆，$\lambda$ 是一个本征值。证毕。

8.D 若尔当形

现在我们终于可以描述幂零算子的精细结构了。

命题 8.55：幂零算子的基 设 $N \in \mathcal{L}(V)$ 是幂零的。则存在 $V$ 的一组基 $(v_1, \ldots, v_n)$ 和正整数 $m_1, \ldots, m_k$ 使得 (a) $(N^{m_1-1}v_1, \ldots, Nv_1, v_1, \ldots, N^{m_k-1}v_k, \ldots, Nv_k, v_k)$ 是 $V$ 的一个基。 (b) $N^{m_1}v_1 = 0, \ldots, N^{m_k}v_k = 0$。

在这个基下，$N$ 的矩阵具有一种非常特殊的块对角形式，每个块被称为一个若尔当块，其对角线为0，对角线上方为1。

结合定理 8.21，我们可以将任何算子 $T$ 分解为 $T = \lambda_j I + N_j$ 在每个广义本征空间上的形式。对每个幂零算子 $N_j$ 应用上述命题，我们就可以得到 $T$ 的一个基，使得其矩阵呈现块对角形式。

定义 8.59：若尔当基 (Jordan basis) $T \in \mathcal{L}(V)$ 的一个基称为 $T$ 的若尔当基，如果 $T$ 关于这个基的矩阵是块对角的，其中每个块都是形如 $$ A_j = \begin{pmatrix} \lambda_j & 1 & & 0 \\ & \lambda_j & \ddots & \\ & & \ddots & 1 \\ 0 & & & \lambda_j \end{pmatrix} $$ 的上三角矩阵。

定理 8.60：若尔当形 设 $V$ 是一个复向量空间，$T \in \mathcal{L}(V)$。则存在 $V$ 的一个基使得 $T$ 关于该基的矩阵具有若尔当形（即块对角形式，每个块是一个若尔当块）。

这个定理是本章的巅峰，它为任何一个复向量空间上的算子提供了一个“标准”的矩阵表示。这个表示虽然不一定是对角的，但已经足够简单，并揭示了算子的所有重要信息（本征值、代数重数、几何重数等）。

第八讲小结

今天我们深入探讨了一般复向量空间上算子的结构，这是一段充满挑战但回报丰厚的旅程。

  1. 我们引入了广义本征向量和广义本征空间的概念，这是处理不可对角化算子的关键。
  2. 我们证明了核心的分解定理：任何复向量空间都可以分解为其算子所有广义本征空间的直和。这使得我们可以将对一般算子的研究分解为对“纯粹”部分的研究，即 $\lambda I + N$ 的形式，其中 $N$ 是幂零算子。
  3. 我们引入了特征多项式和极小多项式，并证明了深刻的凯莱-哈密顿定理，即任何算子都是其特征多项式的根。
  4. 最终，通过分析幂零算子的结构，我们得出结论：任何复向量空间上的算子都有一个若尔当基，使其矩阵呈现出若尔当形。
